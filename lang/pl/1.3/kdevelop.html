Ta strona zawiera wszystkie informacje o seriach wydaniań KDevelop 1.x

Rozwijanie KDE 1.x zostało zatrzymane 11. Grudnia 2000 wraz z wydaniem KDevelop 1.3, nie planuje się więcej wydań KDevelop 1.x dla KDE 1.x.
KDevelop 1.4 został zrobiony dla KDE 2.x i Qt 2.x. ale radzimy używać KDevelop 2.1.5 jeśli potrzebujesz KDevelop dla KDE 2.x.
Obecnie pozostały nam 4 życzenia w KDevelop 3.0 (dla KDE 3.0, Qt 3.0) z wielu błędów i życzeń KDevelop 1.3.

Kompilowanie wersji 1.3 z repozytorium CVS

Aby dostać własnoręcznie-skompilowaną wersję KDevelop 1.3 ze źródeł cvs, sprawdź moduł CVS "kdevelop" na :pserver:anonymous@anoncvs.kde.org:/home/kde używając poprawionej gałęzi CVS KDEVELOP_1_0

komendy jeśli używasz KDE 1.x:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Aby skompilowac i zainstalować KDevelop (wersja CVS) w swoim systemie,
wpisz poniższe komendy w katalogu głównym KDevelop:

wersja kompilowana pod wydanie: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

wersja kompilowana pod debugowanie: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)
