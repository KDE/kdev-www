  • KDevelop menambahkan tiga skrip tambahan untuk membuat hidup anda jauh lebih mudah; ini telah terinstal di $prefix/bin (usually $KDEDIR/bin):
    • extractrc: skrip ini asalnya berada di modul kdesdk dari KDE CVS (KDE Software Development Kit). Ini membuat otomatis pot-file untuk aplikasi anda dan mengekstrak string dari file sumber XML yang digunakan untuk membuat aplikasi GUI di KDE 2.0
    • kimport: kimport adalah skrip perl yang memungkinkan anda untuk membuat *.kdevprj (KDevelp Project File) dari hampir semua automake/autoconf yang kompatibel dengan proyek yang telah ada sebelumnya. Gunakan: lihat ke direktori proyek anda dan panggil kimport > <namaproyekanda>.kdevprj. Ini akan memilihl output ke projectfile yang anda bisa load ke dalam KDevelop
    • kdevdlg2ui: merubah file *.kdevdlg yang telah ada dan yang dibuat oleh KDevelops Dialogeditor ke dalam file *.ui yang bisa dirubah dengan Qts Designer
