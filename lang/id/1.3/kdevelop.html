Halaman ini memfokuskan ke semua informasi tentang KDevelop 1.x rilis

Pengembangan KDE 1.x sudah diberhentikan pada tanggal 11 Desember 2000 dengan KDevelop 1.3 rilis, dan tidak ada rencana untuk melanjutkan perkembangan KDevelop 1.x untuk KDE 1.x.
KDevelop 1.4 dibuat untuk KDE 2.x dan Qt 2.x. tapi tolong gunakan KDevelop 2.1.5 kalau anda perlu KDevelop untuk KDE 2.x.
Sementara ini hanya tersisa empat permohonan di KDevelop 3.0 (untuk KDE 3.0, Qt 3.0) dari banyaknya bugs dan permohonan untuk KDevelop 1.3.

Meng-kompile versi 1.3 dari gudang CVS (CVS repository)

Untuk mendapatkan kompile-sendiri (self-compiled) KDevelop 1.3 dari source cvs, check module CVS "kdevelop" di :pserver:anonymous@anoncvs.kde.org:/home/kde dengan menggunakan the bugfix CVS branch KDEVELOP_1_0

command line kalau anda menggunakan KDE 1.x:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Dengan tujuan untuk meng-kompile dan meng-install KDevelop (versi CVS) di sistem anda,
ketikan perintah-perintah (commands) dibawah ini di base directory KDevelop distribution:

versi rilis-kompilasi: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

versi debug-kompilasi: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)

Bugs yang sudah diketahui
