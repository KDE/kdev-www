Binari :

Beberapa vendor Linux/UNIX OS sudah berbaik hati untuk menyediakan paket binari KDevelop untuk beberapa versi distribusi mereka. Beberapa sukarelawan juga sudah melakukan hal yang sama. Beberapa dari paket binari ini bisa di download dengan gratis dari KDE http atau FTP.



Kode asal untuk KDE >= :


  1. Sebelum dilanjutkan, tolong periksa dulu kalau software-software yang dibutuhkan sudah di install di komputer anda.

  2. Buka kode asal tersebut, lalu masuk ke base directory:
    $ bunzip2 kdevelop-.tar.bz2
    $ tar -xf kdevelop-.tar
    $ cd kdevelop-
