KDE IDE yang gratis untuk Pengembangan dalam Perusahaan

KDE Memasukan KDevelop, IDE terbaik untuk Linux/UNIX, sebagai bagian dari KDE 2.2

15 Agustus 2001 (INTERNET). Proyek KDE hari diumumkan akan segera dirilis KDE 2.2, yang sangat baik, desktop Internet-enabled yang mudah digunakan Lingkungan Pengembangan yang Terintegrasi (IDE)untuk Linux. KDE 2.2 disalurkan dengan KDevelop 2.0. KDevelop adalah lingkungan pengembangan terintegrasi (IDE) Linux/UNIX terbaik untuk untuk pengembangan C/C++. KDevelop menyediakan program yang profesional dengan alat-alat yang sangat baik untuk pengembangan yang cepat dan desain aplikasi C/C++ yang sangat mutakhir.

Consistent with KDE's rapid and disciplined development pace, the release of KDevelop features a large number of speed/performance enhancements, feature additions and stability improvements. Among those are: a redesigned user interface offering three different working modes, including multiple views per document; a new setup wizard for easy configuration; and additional enhancements that instantly improve a developer's overall productivity, such as configurable sound notifications and text-colored compiler output. In addition, KDevelop 2.0 adds support for the Qt 3.0 beta releases, together with the accompanying Qt Designer and Qt Linguist tools.

KDE dan semua komponen-nya, termasuk KDevelop, telah tersedia gratis dibawah lisensi Open Source KDE server dan mirrors dan juga dapat terdapat dalam CD-ROM.

"Terima kasih kepada kualitas yang sangat baik dari pengembangan framework KDE 2 dan tanggapan yang sangat berharga dari para pengguna kami, kami memberikan pengalaman desktop yang lebih baik, terintegrasi dan banyak fitur dalam waktu yang singkat. Dengan penundaan rilis KOffice yang stabil bulan ini, KDE menawarkan kualitas lengkap dan terbaik, dan pengembangan platform tanpa biaya dan pelarangan terhadap desktop komersial," kata Waldo Bastian, manajer rilis untuk KDE 2.2 dan SuSE Labs developer.

"Karena komitmen kami yang kuat untuk KDE dan KDevelop, kami juga membangkitkan semangat untuk memasukkan desktop dan lingkungan pengambangan yang memungkinkan perusahaan bermigrasi ke aplikasi dan desktop dengan platform KDE/Qt. KDE's mudah digunakan dan para pengembang akan membuktikannya di lapangan, dan mengesahkan pilihan desktop KDE sebagai yang terbaik dan penggunaan biaya terefektif untuk penggunaan perusahaan besar," komentar dari Ralf Nolden, pengurus KDevelop dan pengembang KDE.

"Ini sangat penting untuk perusahaan untuk mengerti bahwa penghematan biaya yang sangat besar mereka dapat migrasi ke KDE dan tidak membutuhkan pengorbanan dalam persetujuan software kepada para penggunanya," ditambahkan oleh Andreas Pour, Ketua dari KDE League. " kapabilitas dari Konqueror, perluasan kesatuan free software terdapat di KDE/Qt dan banyak proyek Open Source lainnya akan memecahkan banyak masalah. Untuk aplikasi ini juga tersedia untuk Windows pada saat ini, beberapa produk alternatif, keduanya adalah produk komersial dan Open Source, yang terdiri dari: berjalan pada terminal Windows terminal di KDE, aplikasi Windows dapat berjalan dalam KDE/Linux, dan menyediakan sesi remote desktop. Tidak ada alasan perusahaan tidak membuka diri mereka setidaknya sebagian dari biaya lisensi untuk para pengguna desktop mereka."

KDE 2: K Development Environment. KDE 2.2 menawarkan kepada para pengembang dengan IDE yang sangat baik sebagai penyempurnaan teknologi dasar lebih dari seri KDE 1 yang dikritik. Bagian dari terknologi tersebut adalah the Desktop COmmunication Protocol (DCOP), I/O libraries (KIO), komponen model objek (KParts), GUI class berdasarkan XML, dan standards-compliant HTML rendering engine (KHTML).

KDevelop adalah IDE Linux terbaik dengan banyak fitur untuk mengembangkan aplikasi dengan cepat, termasuk pembuatan dialog GUI, debugging terintegrasi, pengelolaan proyek, dokumentasi dan fasilitas terjemahan, dukungan terhadap pengembangan yang telah ada, dan banyak lagi. Rilis ini termasuk banyak fitur baru lainnya, termasuk setup wizard, navigasi kode, console, halaman petunjuk (man page) dan teks yang berwarna, dan banyak template proyek baru, termasuk modul kontrol utama (KControl), applet panel (Kicker), kio-slaves, Konqueror plugins dan desktop themes.

KParts, KDE 2's membuktikan model komponen objek, mengendalikan semua aspek dari aplikasi embedding, seperti memindahkan toolbars dan memasukkan menu yang terkait ketika embedded component telah diaktifkan atau tidak diaktifkan. KParts dapat juga berinteraksi dengan KIO trader untuk mencari handler yang tersedia untuk mimetypes atau services/protocols yang spesifik. Teknologi yang diperluas ini digunakan oleh KOffice suite dan Konqueror.

KIO implementasi aplikasi I/O dalam proses yang terpisah memungkinkan tidak menghambat GUI tanpa penggunaan threads. Kelas adalah network dan protocol yang tansparan dan tetap dapat digunakan untuk mengakses HTTP, FTP, POP, IMAP, NFS, SMB, LDAP dan file-file local. Lebih lanjut, modul dan desain yang dapat diperluas mengijinkan para pengembang untuk "menambahkan" protokol tambahan seperti WebDAV, yang akan secara otomatis tersedia di semua aplikasi KDE. KIO juga mengimplementasikan trader yang bisa mencari pengendali untuk mimetypes yang spesifik; pengendali ini dapat di-embed dengan aplikasi yang diminta menggunakan teknologi KParts.

DCOP komunikasi protokol client-ke-client dilanjutkan oleh server lewat pustaka X11 ICE standar. Protokol ini mendukung kedua pengirim pesan dan panggilan prosedur remote menggunakan XML-RPC ke DCOP "gateway". Bindings untuk C, C++ dan Python, dan juga Java bindings eksperimental, telah tersedia.

KDE juga menyediakan binding dari banyak bahasa. Sebagai contoh, KDE kdejava menyediakan Java bindings secara penuh untuk KDE dan Qt, yang dapat dilihat dan mempunyai persamaan dengan versi C++, termasuk akses untuk sinyal/tempat C++.

Menginstal Paket Binari

Paket Binari untuk KDevelop 2.0. Semua distributor utama Linux dan beberapa distributor Unix telah menyediakan paket binari KDE 2.2/KDevelop 2.0 untuk versi terakhir dari distribusi mereka. Beberapa paket binari tersedia secara gratis untuk di-download di http://master.kde.org/pub/kde/stable/2.2/ atau di direktori yang sama dari banyak server ftp KDE mirrors. Harap dicatat bahwa tim KDE dan KDevelop tidak bertanggung jawab atas paket ini sebagaimana jika disediakan oleh pihak ketiga -- pada umumnya, tapi tidak selalu, distributor dari distribusi yang relevan (jika anda tidak dapat mendapatkan paket binari untuk distribusi anda, harap dibaca Kebijakan Paket Binari KDE).

Kebutuhan Pustaka. Kebutuhan pustaka untuk beberapa paket binari bervariasi dengan sistem dalam paket dimana telah dikompilasi. Harap diingat bahwa beberapa paket binari mungkin memerlukan versi yang lebih baru dari Qt dan pustaka lainnya telah termasuk dengan distribusi yang umum (e.g., LinuxDistro 8.0 mungkin telah dilengkapi dengan qt-2.2.3 tetapi paket dibawah ini mungkin memerlukan qt-2.3.x). Untuk kebutuhan pustaka untuk KDE, harap lihat teks di Kebutuhan Pustaka - Source Code .

Tentang KDE/KDevelop

KDE adalah proyek independen, dan kolaraboratif dari ratusan pengembang di seluruh penjuru dunia yang bekerja lewat internet untuk membuat desktop yang baik, dapat diubah-ubah dan stabil yang berbasiskan komponen, arsitektur network-transparent. KDE adalah bukti pekerjaan nyata dari kekuatan Open Source "Bazaar-style" model pengembangan software untuk membuat teknologi tingkat atas yang sama dan lebih baik dibandingkan dengan software komersial yang sangat kompleks.

Kunjungi web site KDE untuk KDE FAQ, screenshots, informasi KOffice dan informasi pengembang. Lebih lanjut tentang informasi KDE tersedia di KDE's family of web sites.

Informasi tentang KDevelop tersedia di website-nya, termasuk screenshots (halaman 1, halaman 2, halaman 3); a Petunjuk Penggunaan; panduan (KDevelop dan KDE/Qt); dan forum diskusi.
