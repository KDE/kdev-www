31 Agustus 2003 - Konferensi telai usai dan KDevelop telah melihat penyempurnaan - C++ parser sekarang lebih cepat, dapat menangani completion untuk Qt's SIGNAL dan SLOT, interface untuk para pengguna juga telah disempurnakan, semua plugin telah mendapat nomor versi untuk mencegah DLL hell, bahasa pemrograman yang aktif dapat bisa diubah sesuai permintaan, dukungan terhadap SQL mempunyai class-view dan pekerjaan dimulai dalam proyek manager generic. Slide presentasi oleh harryF's dapat ditemukan di disini.
