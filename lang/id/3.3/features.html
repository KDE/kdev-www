Termasuk semua yang terdapat dalam fitur KDevelop 3.2.x ditambah dengan:
  • Fitur baru dalam bagian KDevelop Rencana fitur tambahan dalam KDE 3.5
  • Anda sekarang dapat menggunakan CMAKE untuk meng-compile KDevelop
  • Dukungan terhadapa bahasa C++ yang diperbaharui:
    • Dukungan untuk QT4, kode yang berbeda telah dibuat untuk proyek Qt3.x and Qt4.x
    • Penambahan tab baru dalam C++ dialog option untuk menyusun librari Qt yang digunakan. Anda dapat menyusun Qt yang dipergunakan, dengan versi utama dan dimana librari Qt telah di-instal.
  • Pernyempurnaan debugger untuk C, C++ dan bahasa Ruby:
    • Jendela breakpoint debugger membuat semakin mudah untuk digunakan
    • Memperlihatkan secara otomatis jendela variabel ketika aplikasi berhenti, dan menyembunyikan aplikasi ketika berhenti.
    • Implementasi ekspresi/variabel editing dalam jendela variabel.
    • Penyempurnaan cara kerja variabel
  • Dukungan bahasa PHP5
  • Manajer proyek generik yang baru:
    • Terbuat dari arsitektur platform kdevelop dimana manajemen proyek GUI adalah abstrak dari spesifik buildsystem. Buildsystem ditangani menggunakan plugin ("proyek importer, builder and editor")
    • Dukungan workspace telah tersedia
    • Proyek builder: make builder
    • Proyek importer: automake, custom makefiles, qmake
  • lihat halaman perubahan log untuk lebih jelas
