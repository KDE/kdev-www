Halaman ini berisi informasi tentang KDevelop seri rilis

Versi .0 telah dirilis dengan keluarnya KDE-.0. KDevelop .x bugfix versions akan dirilis bersamaan dengan KDE-.x

On 2007-01-25 KDevelop 3.3.6, the last version of this 3.3.x series, was released. This series is no longer developed nor maintained. Please use version 3.4.x instead.
