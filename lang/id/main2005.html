21. September 2005 - KDevelop 3.3 beta 1 (aka 3.2.91) got released to the public together with KDE 3.5 beta 1. You can download it and see its Changelog.

11. August 2005 - KDevelop 3.3 alpha 1 (aka 3.2.90) got released to the public together with KDE 3.5 alpha 1. You can download it and see its Changelog. KDevelop 3.3 runs on KDE >= 3.2 and QT >= 3.2 but can be used to develop both QT3 and QT4 appplications.
KDevelop HEAD (SVN trunk) now requires QT4 and is planed for release as KDevelop 4.0 together with KDE 4.0 by the end of next year.

It has been translated to 19 languages, the user manual has been translated to 7 languages and this website is available in 12 languages.

13 Januari 2005 - KDevelop 3.2 beta 1 also known as 3.1.91 got released to the public. You can download it and see it's Changelog. Future KDevelop 3.2.x releases will follow the KDE 3.4 Release Plan.
This is also the first release cycle of the new KDevelop Platform API and the first release where the website HEAD/ files are brancheable. This means less work for both the translators and the webmaster. The translators only need to translate HEAD/ files and the site creates the stable branch files using some PHP magic.
