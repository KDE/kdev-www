Справочник KDevelop Platform Libraries API
Следующее поколение KDevelop будет называться KDevelop 4.0. Оно будет основано на Qt 4.3 и KDE 4.0. Для того чтобы было проще использовать возможности KDevelop при разработке собственных приложений типа IDE, мы разделили KDevelop на библиотеку - Платформа KDevelop (KDevelop Platform) - и само приложение - KDevelop. Таким образом, такие приложения как kile, quanta и др. смогут совместно с KDevelop повторно использовать один и тот же код. Советуем ознакомиться со справочником KDevelop Platform Libraries API перед тем как начать разрабоку своего нового приложения.
