Заметки:
  1. Только PO каталоги, имеющие соответствующие POT файлы включены в статистику.
  2. Итоги подсчитываются относительно количества сообщений в PO файлах. Например, если по ошибке переведенные PO файлы имеют отличное от POT число сообщений, то возможны расхождения.
  3. Строка с красным фоном указывает на то, что команда переводчиков имеет хотя бы один PO файл с хотя бы одной ошибкой.
  4. Строка с голубым фоном указывает на отсутствие POT файлов для имеющихся PO файлов.
  5. Возможно, что итоговые суммы не будут точно равны 100% вследствие ошибок округления.

Примечание:
   - переведенные сообщения
   - ошибочные собщения
   - непереведенные сообщения
   - информация недоступна
