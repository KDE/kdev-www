Пакеты для платформы :

Некоторые поставщики Linux/UNIX OS предоставляют бинарные пакеты KDevelop для определенных версий их дистрибутивов, в других случаях пакеты собраны волонтерами. Некоторые из этих бинарных пакетов доступны с с зеркал KDE http или FTP.



Исходный код для KDE >= 3.2 :


  1. Перед тем как продолжать, проверьте наличие необходимого ПО.

  2. Распакуйте исходники:
    $ bunzip2 kdevelop-3.1.x.tar.bz2
    $ tar -xf kdevelop-3.1.x.tar
    $ cd kdevelop-3.1.x
