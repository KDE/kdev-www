Содержит все возможности KDevelop 3.0.x плюс:
  • Шаблоны кода
    • SnippetPart добавляет панель, прикрипленную справа в окне KDevelop
    • Добавление, редактирование и удаление шаблонов с помощью всплывающего меню
    • Двойной щелчок на шаблоне вставляет его в активное окно редактора в текущую позицию курсора
    • Подсказки отображают содержимое шаблона
    • Шаблоны сохраняются в домашнем каталоге пользователя, таким образом каждый пользователь может иметь свои шаблоны
    • Шаблоны могут содержать переменные в форме $VARNAME$. При использовании шаблона будет предложено заполнить переменные значениями
  • Интеграция Doxygen (Генератор документации по API)
    • Поддерживает все языки программирования, что и doxygen (кроме IDL и D)
    • Обеспечивает предпросмотр документации, которую может сгенерировать doxygen для текущего файла
    • Конфигурирует doxygen
    • Запускает doxygen
    • Обеспечивает просмотр сгенерированной doxygen документации (в форматах html, man или pdf)
    • Может использовать генерируемый doxygen'ом XML для подсказок при наборе кода
  • Модульный просмотрщик документации
    • Предоставляет унифицированный интерфейс к просмотру документации.
    • Имеется содержание, индекс и полнотекстовый поиск (с htdig).
    • Автоопределение и включение документации Qt, KDE Libs и DevHelp.
    • Поддержка других коллекций документации (в формате KDevelopTOC).
    • Возможность открытия локальной, удаленной и даже архивированной документации (используя любой протокол, поддерживаемый Konqueror - http, ftp, sftp, tar, zip, и т.д.).
    • Поддерживает форматы TXT, HTML, DevHelp, CHM, PDF, Djvu и PDB.
    • Может использовать любой формат, поддерживаемый KDE
  • Улучшенный мастер создания приложений
    • Мастер создания приложений генерирует полноценные, готовые к запуску приложения.
    • Существуют шаблоны для каждого поддерживаемого языка и системы сборки.
    • Возможность использования шаблонов приложений, не включенных в дистрибутив.
    • Содержит все шаблоны из KDevelop 3.0 + шаблоны приложений GTK, GNOME и Bonobo
    • Поддерживает возможности, о которых знает только Ian Geiser :)
  • Ориентация на RAD (Rapid Application Development)
    • Новое приложение - kdevdesigner позволяет интегрировать Qt Designer в KDevelop, обеспечивая быструю разработку интерфейсов для Qt/KDE.
    • Новое приложение - kdevassistant обеспечивает просмотр документации в стиле QT Assistant.
    • Оба этих приложения могут быть использованы как по отдельности, так и из KDevelop, т.е. предоставляется один инструмент для решения всех задач разработчика.
