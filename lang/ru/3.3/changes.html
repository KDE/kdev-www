Поддерживает KDE >= . Была помечена 26.07.2005, но не была выпущена. Включает все новые возможности KDevelop , а также следующие измнения:
  • NEW - Описание возможностей в секции KDevelop на странице KDE 3.5 Feature Plan
  • NEW - Теперь можно использовать CMAKE для компиляции KDevelop
  • IMPROVED - Поддержка C++:
    • Поддержка для QT4
    • Добавлена новая вкладка в диалоге опций C++ для настройки используемой версии библиотеки Qt. Вы можете настроить какую версию использовать и указать путь к библиотеке Qt (QTDIR) для своего проекта.
    • Мастер создания новых классов - Чекбоксы для Qt заблокированы для проектов не использующих Qt. Различный код генерируется для Qt3.x и Qt4.x проектов.
  • IMPROVED - Отладчик:
    • Окно с точками останова стало проще использовать
    • Исправлена ошибка с отключенными точками останова
    • Автоматический показ переменных после остановки программы и сокрытие после завершения программы.
    • Добавлено редактирование значений выражений/переменных в окне просмотра переменных.
    • Улучшен просмотр переменных
  • NEW - Поддержка для PHP5 (поиск исполняемого модуля php)
  • IMPROVED - Окно вывода приложения:
    • Оптимизация по скорости
  • IMPROVED - Общий менеджер проектов:
    • Основан на архитектуре платформы KDevelop, где графический интерфей управления проектом независит от определенного средства сборки. Средства сборки поддерживаются как подключаемые модули ("Средства импорта, сборки и редакторы")
    • Поддержка рабочего пространства
    • Сборка проектов: make
    • Импорт проектов: automake, пользовательские makefile'ы, qmake
  • IMPROVED - Руководство пользователя
  • FIX - Около 48 ошибок версии KDevelop 3.2.x
