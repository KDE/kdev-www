Содержит возможности KDevelop 3.2.x плюс:
  • Список новых возможностей KDevelop добавлен в раздел KDE 3.5 Feature Plan
  • Теперь можно использовать CMAKE для компиляции KDevelop
  • Улучшена поддержка C++:
    • Реализована поддержка QT4. Генерируется различный код для Qt3.x и Qt4.x проектов
    • Добавлена новая вкладка в диалоге параметров C++ для настройки используемой библиотеки Qt. Можно указать какую версию использовать и где установлена библиотека.
  • Улучшен отладчик для C, C++ и Ruby:
    • Стало проще использование окна со списком точек останова
    • Автоматический показ переменных при прерывании выполнения приложения, и сокрытие при завершении приложения.
    • Реализация редактирования выражения/переменной в окне просмотра переменных.
    • Улучшен просмотр переменных
  • Добавлена поддержка для PHP5
  • Новые средства управления проектом:
    • Основанный на архитектуре платформы kdevelop в которой средства управления проектом отделены от средств управления проектом. Средства сборки поддерживаются в виде подключаемых модулей ("импорт проектов, средства сборки и редакторы")
    • Поддержка рабочего пространства
    • Средства сборки: make
    • Импорт проектов: automake, обычный makefile, qmake
  • Смотрите более подробный список изменений
