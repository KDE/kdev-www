С февраля 1999 работает список рассылки (kdevelop at kdevelop.org) с 600-700 подписчиками.. Этот список предназначен для обсуждения вопросов, связанных с KDevelop.
Архив списка: barney.cs.uni-potsdam.de
Благодарности Hank Leininger.

Есть два списка рассылки: kdevelop-devel для разработчиков KDevelop и kdevelop-bugs, в который отсылается информация об изменениях в базе данных ошибок с bugs.kde.org. Можно подписаться и отменить подписку на эти списки рассылки с помощью формы вверху этой страницы.

Сообщения из этих списков рассылки могут быть найдены: в архиве рассылки kdevelop-devel и в архиве рассылки kdevelop-bugs.

Можно также подписаться на рассылку с последними изменениями в KDevelop SVN.
