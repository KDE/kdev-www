Пакеты для платформы :

Некоторые поставщики Linux/UNIX OS предоставляют бинарные пакеты KDevelop для определенных версий их дистрибутивов, в других случаях пакеты собраны волонтерами. Некоторые из этих бинарных пакетов доступны с зеркал KDE http или FTP.

       $ tar -xf kdevelop-3.0.x.tar
       $ cd kdevelop-3.0.x
