Это также первая версия в рамках нового KDevelop Platform API и первая версия где файлы web-сайта из HEAD/ ветвятся. Это означает меньше работы как для переводчиков, так и для web-мастера. Переводчикам теперь требуется только переводить файлы из HEAD/ и сайт используя немного PHP волшебства создаст файлы стабильной ветки.
