Diese Seite fasst alle Informationen über die KDevelop 1.x Serie zusammen

Die Entwicklung für KDE 1.x wurde am 11. Dezember 2000 mit der Veröffentlichung von KDevelop 1.3 aufgegeben, und keine weiteren KDevelop 1.x Versionen sind für KDE 1.x geplant.
KDevelop 1.4 ist für KDE 2.x und Qt 2.x, aber bitte nutzen Sie stattdessen KDevelop 2.1.5 falls Sie KDevelop für KDE 2.x benötigen.
Zur Zeit verbleiben nur vier Wünsche in KDevelop 3.0 (für KDE 3.0, Qt 3.0) von den vielen Fehlern und Wünschen von KDevelop 1.3.

Version 1.3 vom CVS Verzeichnis kompilieren

Um ein selbstkompiliertes KDevelop 1.3 aus den CVS Quelltexten zu erhalten, besorgen Sie sich den Zweig KDEVELOP_1_0 des CVS Moduls "kdevelop" unter :pserver:anonymous@anoncvs.kde.org:/home/kde

Kommandozeile falls Sie KDE 1.x benutzen:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Um KDevelop (CVS Version) auf Ihrem System zu kompilieren und zu installieren, führen Sie folgende Befehle im Wurzelverzeichnis Ihrer KDevelop Distribution aus:

eine für die Veröffentlichung kompilierte Version: 
       $ export KDEDIR=/wo/Ihr/KDE1/ist
       $ export QTDIR=/wo/Ihr/qt-1/ist
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

eine mit Debuginformationen kompilierte Version: 
       $ export KDEDIR=/wo/Ihr/KDE1/ist
       $ export QTDIR=/wo/Ihr/qt-1/ist
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)
