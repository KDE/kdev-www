3. FEBRUAR 2004

ZUR SOFORTIGEN VERÖFFENTLICHUNG

KDevelop 3.0 freigegeben

Das KDevelop Team veröffentlicht eine neue Version der preisgekröhnten IDE mit vielen neuen Features

3. Februar 2004 (The INTERNET). Das KDevelop Team freut sich, KDevelop 3.0, das erste größere Release der komplett neugeschriebenen, preisgekröhnten Integrierten Entwicklungsumgebung (IDE) freigeben zu können.

"KDevelop ist der Schlüssel zu neuen Leveln der Produktivität. Ob Systemprogrammierung oder GUI RAD Entwicklung, die IDE passt sich jedem Zweck an. Durch die Integration aller unter UNIX üblichen Entwicklungswerkzeuge, die direkte Anbindung an verschiedene Versionskontrollsysteme, die die Arbeit im Team merklich erleichtern, und die große Anzahl von unterstützten Sprachen ist KDevelop eine exzellente Wahl für alle UNIX Entwickler. Durch die Verfügbarkeit auf unterschiedlichsten Systemplattformen wie Mac OS X und Windows/Cgywin stellt KDevelop 3.0 ein ideales Werkzeug für alle Multiplattform-Entwickler dar. Die Unterstützung bekannter Entwicklungswerkzeuge ermöglicht die Entwicklung auf einer Vielzahl von Plattformen, inklusive eingebetteten Systemen." hebt Alexander Dymo, Mitverantwortlicher im Projekt, hervor.

"Mit seinem neuen sprach-unabhängigen, Plugin basierendem Kern und weitgestreuten Unterstützung für bekannte Entwicklungswerkzeuge ist KDevelop bereits der de-facto Standard im Bereich IDEs unter Linux/Unix" fügt Harald Fernengel, ein langjähriges Projektmitglied, hinzu.

"Mit dieser Veröffentlichung legt das KDevelop Team eine bewundernswerte Leistung an den Tag. Über 40 Entwickler trugen über einen Entwicklungszeitraum von 14 Monaten zum Gelingen des Projekts bei. Besondere Sorgfalt wurde der Dokumentation und der grafischen Oberfläche gewidmet. Diese Version ist mächtig, und dennoch einfach zu benutzen." bemerkt Amilcar do Carmo Lucas, Veröffentlichungs-Mitkoordinator.

Größere Erweiterungen

KDevelop unterstützt jetzt 15 Programmiersprachen und eine Vielzahl an Versionskontrollsystemen, Debuggern, Dokumentationsformaten und Compilern. Jeder Editor der die KTextEditor Schnittstelle unterstützt kann als Editor für KDevelop verwendet werden, inklusive kate, QEditor und kvim. Hilfsprogramme wie automake, qmake, und weitere werden direkt unterstützt. Das bedeutet, dass Änderungen an den nativen Projektdateien automatisch in KDevelop übernommen werden und anders herum.

Die C++ Unterstützung bietet eine Klassenansicht die alle Symbole entweder flach oder hierarchisch nach Namensränen anordnet und eine einfache Navigation im Code ermöglicht. Ein Parser im Hintergrund aktualisiert die Symbole stetig und zeigt Syntaxfehler im Quellcode an. Die Codevervollständigung funktioniert sogar mit Qt Signalen und Slots.

Durch die neue, offene Architektur kann KDevelop in jeder Hinsicht an die individuellen Bedürfnisse angepasst werden. Die Benutzeroberfläche kann per Mausklick von der klassischen MDI-Ansicht in die moderne IDEA1 Ansicht, die den Code in den Mittelpunkt stellt und Werkzeuge bei Bedarf darstellt, umgeschaltet werden.

KDevelop bietet weitreichende Unterstützung für die Navigation im Code an. Der Zugriff auf Symbole und Dateinamen ist durch das Eintippen eines Teils des Namens möglich, vor kurzem geänderte Dateien können mit einem einfachen Tastendruck erreicht werden.

KDevelop bietet Dokumentationsgeneratoren, Code Debugger, Speicher-Checker, Lesezeichenmanagement, und dutzende weitere Werkzeuge die die Entwicklung einfach und komfortabel machen. Es wurde in mehr als 16 Sprachen von Englisch bis Tamil übersetzt.

Installation der KDevelop 3.0 Binärpakete

Unter http://www.kdevelop.org/index.html?filename=3.0/download.html gibt es Binärpakete für alle größeren Distributionen.

Eine Alpha-Version eines Binärpakets für Mac OS X befindet sich unter http://kde.opendarwin.org.

Das kde-cygwin Projekt bietet KDevelop unter http://kde-cygwin.sourceforge.net/kde3/installation.php an.

Bitte beachten Sie, dass diese Binärpakete nicht vom KDevelop Team, sondern von Dritten, bereitgestellt werden.

KDevelop 3.0 compilieren

Der komplette Quellcode von KDevelop 3.0 kann unter http://www.kdevelop.org kostenlos heruntergeladen werden. Ausführliche Informationen zum Übersetzen der Quelltexte befinden sich unter http://www.kdevelop.org/index.html?filename=3.0/branches_compiling.html.

Über KDevelop

KDevelop ist ein unabhängiges Projekt mit dem Ziel eine mächtige und einfach zu benutzende integrierte Entwicklungsumgebung (IDE) zu entwickeln. Als Teil des KDE Projekts benutzt KDevelop die gleiche offene Entwicklungsmodell und das gleiche modulare Framework. Das Ergebnis ist eine höchst anpassbare, sprachunabhängige IDE unter der GPL Lizenz, vorangetrieben bei vielen Entwicklern weltweit. Mehr Informationen über KDevelop befinden sich auf der Homepage unter http://www.kdevelop.org.

Über KDE

KDE ist ein unabhängiges Projekt von Entwicklern, Übersetzern, Designern und anderen Experten, die über das Internet zusammenarbeiten um eine anspruchsvolle, anpassbare und stabile Desktop- und Büroumgebung durch eine flexible, modular aufgebaute und netzwerktransparente Architektur zu schaffen. KDE bietet eine stabile, ausgereifte Arbeitsoberfläche, eine vollständige und modular aufgebaute Büro-Suite (KOffice), eine große Anzahl von Netzwerk- und Entwicklungswerkzeugen und Hilfsprogrammen sowie mit KDevelop eine effiziente, intuitive Entwicklungsumgebung. KDE ist der lebende Beweis, dass basarähnliche Open Source Entwicklung hochqualitative Anwendungen produzieren kann, die den Vergleich mit komplexen kommerziellen Programmen nicht scheuen braucht.
