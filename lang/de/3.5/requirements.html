Bevor Sie mit dem Kompilieren Ihrer frisch heruntergeladenen KDevelop-Version starten, versichern Sie sich, dass Sie die folgenden Programme installiert haben. Beachten Sie, dass Sie seltsame Fehler erhalten können, wenn Sie ältere Versionen als die angegebenen verwenden.
Normalerweise können Sie sehen, welche Version einer Software auf ihrem Rechner installiert ist, indem sie das Programm mit dem Argument "--version" aufrufen. Um z. B. herauszufinden welches autoconf Sie auf Ihrem Rechner installiert haben, geben Sie folgendes ein: 
   autoconf --version

Bitte beachten Sie, dass Sie zur Benutzung der Sprachenunterstützung für Ada, Pascal, Java etc. der entsprechende Kompiler vorhanden sein muss. KDevelop begrenzt nicht die Auswahl der Kompiler, liefert aber für einige von ihnen Konfigurationsdialoge als Plug-In mit. Zur Zeit vorhandene Plug-Ins sind gcc, g++, f77, pghpf (Portland Group High Performance Fortran Compiler), pgf77 (Portland Group Fortran 77 Compiler), dcc (Borland Delphi Compiler) und fpc (Free Pascal Compiler).
