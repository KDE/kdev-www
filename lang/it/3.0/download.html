Binari :

Alchuni fornitori di SO Linux/UNIX hanno gentilmente fornito pacchetti binari di KDevelop per alcune versioni delle loro distribuzioni, ed in altri casi i volontari della comunita` hanno fatto lo stesso. Alcuni di questi binari sono disponibili per essere scaricati liberamente dai mirror http ed ftp di KDE.



Sorgenti per KDE >= 3.0 :


Prima di continuare verifica che sul tuo computer sia installato il software necessario.

Decomprimi i file sorgente ed entra nella directory base: 
       $ bunzip2 kdevelop-3.0.x.tar.bz2
       $ tar -xf kdevelop-3.0.x.tar
       $ cd kdevelop-3.0.x
