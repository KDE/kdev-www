Questa pagina riunisce tutte le informazioni relative alla serie 1.x di KDevelop

Lo sviluppo per KDE 1.x è terminato l'11 Dicembre 2000 con il rilascio di KDevelop 1.3, e non sono previste ulteriori versioni della serie KDevelop 1.x per KDE 1.x.
KDevelop 1.4 è progettato per KDE 2.x e Qt 2.x., ma è preferibile che usiate KDevelop 2.1.5 se avete necessità di una versione di KDevelop per KDE 2.x.
Attualmente solo quattro desideri rimangono in KDevelop 3.0 (per KDE 3.0, Qt 3.0) dai molti bugs e desideri di KDevelop 1.3.

Plugin e patch

Per differenti ragioni (per lo più legate alla stabilità), alcune patch più grandi non possono essere applicate alla versione attuale. Esse, tuttavia, offrono notevoli miglioramenti, così potete scaricarle da qui.
Ricordate che queste sono patch per i sorgenti, quindi dovete scaricare un pacchetto contenente i sorgenti adeguati, applicare loro le patch e ricompilarli.

Compilare la versione 1.3 da un repository CVS

Per ottenere un KDevelop 1.3 auto-compilato dai sorgenti cvs, verificate il modulo CVS "kdevelop" su :pserver:anonymous@anoncvs.kde.org:/home/kde usando il ramo KDEVELOP_1_0 del bugfix CVS

da linea di comando se state utilizzando KDE 1.x:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Per compilare ed installare KDevelop (versione-CVS) sul vostro sistema,
eseguite i seguenti comandi nella cartella base della distribuzione KDevelop:

una versione compilata per il rilascio: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

una versione compilata per il debug: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)

Bugs conosciuti
