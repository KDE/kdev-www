Note:
  1. Solo cataloghi PO che hanno un corrispondente file POT sono inclusi nella statistica.
  2. Totale conteggio su numero di messaggi trovati nei files PO. Per esempio, se, per errore, i PO tradotti hanno numeri di messaggi diversi dai corrispondenti POT potresti osservare delle differenze.
  3. Una riga con sfondo rosso vuol dire che il team corrispondente ha un file PO con almeno un errore.
  4. Una riga con sfondo blu vuol dire che il corrispondente file PO non ha un file POT associato.
  5. E' possibile che le somme delle percentuali per tradotti, fuzzy, non tradotti siano superiori o inferiori al 100%. Questo è normale a causa degli errori di arrotondamento quando i numeri sono troncati alla seconda cifra decimale.
