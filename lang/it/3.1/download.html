Binari :

Alcuni venditori di sistemi Linux/UNIX hanno gentilmente fornito pacchetti binari di KDevelop per alcune versioni delle loro distribuzioni, e in altri casi volontari hanno fatto altrettanto. Alcuni di questi pacchetti binari sono disponibili per il download libero via http da KDE o dai mirror FTP.



Sorgenti per KDE >= 3.2 :


  1. Prima di continuare per favore controlla se il tuo computer ha il software necessario installato.

  2. Decomprimi il file sorgente e spostati nella directory di base:
    $ bunzip2 kdevelop-3.1.x.tar.bz2
    $ tar -xf kdevelop-3.1.x.tar
    $ cd kdevelop-3.1.x
