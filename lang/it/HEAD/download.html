Sorgenti per KDE >= :

Ultima versione di sviluppo e quindi instable, snapshot (tar.bz2)

  1. Prima di continuare controlla di avere installato il software richiesto.

  2. Scarica il file sorgente tar.bz2.

  3. Decomprimi il file sorgente e spostati sulla directory di base:
    $ bunzip2 kdevelop.tar.bz2
    $ tar -xf kdevelop.tar
    $ cd kdevelop
