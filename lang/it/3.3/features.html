Contiene tutte le  features di KDevelop 3.2.x  insieme a:
  • Nuove features nella sezione relativa a KDevelop sul Piano delle Features di KDE 3.5
  • Ora puoi usare CMAKE per compilare KDevelop
  • E` stato migliorato il supporto al linguaggio C++:
    • supporto per le QT4, codice differente viene generato per i progetti relativi a Qt3.x e Qt4.x
    • Aggiunti nuovi tab nelle opzioni del C++ per configurare la libreria Qt usata. Puoi scegleire se usare le Qt, quale versione e dov'e` stata installata.
  • E` stato migliorato il debugger per C, C++ e Ruby:
    • La finestra dei breakpoints e` stata resa piu` semplice da usare
    • La finestra delle variabili viene visualizzata non appena l'applicaizone si ferma, e viene chiusa quand l'applicazione termina.
    • Implementato la modifica dei valori delle espressioni/variabili direttamente nella finestra delle variabili.
    • Migliorata la vista delle variabili
  • Supporto per PHP5
  • Nuovo gestore dei progetti generici:
    • E` basato sull'archietettura di piattaforma di kdevelop dove la GUI della gestione del progetto viene astratta da uno specifico metodo di compilazione. I metodi di compilazione sono gestiti mediante plugins ("importazioni del progetto, compilatori e editor")
    • Supporto per il workspace ora disponibile
    • Compilatori del progetto: make builder
    • Importatori del progetto: automake, custom makefiles, qmake
  • leggi la pagina dei  changelog per ulteriori dettagli
