Verze .0 byla vydána s KDE-.0. Opravné verze KDevelop .x budou vydávány společně s KDE-.x

On 2007-01-25 KDevelop 3.3.6, the last version of this 3.3.x series, was released. This series is no longer developed nor maintained. Please use version 3.4.x instead.
