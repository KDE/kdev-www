Předtím, než začnete kompilovat čerstvě staženou verzi KDevelop, ujistěte se, že máte nainstalované následující programy. Mějte na paměti, že se můžete dočkat podivných výsledků, jestliže používáte starší verze.
Obvykle se verzi programu, kterou používáte, dozvíte spuštěním programu s parametrem "--version". Např. verzi autoconf, kterou máte nainstalovanou, zjistíte následovně: 
   autoconf --version

Program/Knihovna Zdroj
KDE >= 3.2.0 www.kde.org
Qt >= 3.2.3 www.trolltech.com/products/qt
g++ >= 2.95.3 (nebo kompatibilní) gcc.gnu.org
GNU make www.gnu.org/software/make
perl >= 5.004 www.perl.com
autoconf >= 2.52 www.gnu.org/software/autoconf
automake >= 1.6 www.gnu.org/software/automake
flex >= 2.5.4 www.gnu.org/software/flex

Abyste mohli KDevelop použít pro projekty psané v jazycích jako jsou ada, pascal, java atd. budete potřebovat odpovídající překladač. KDevelop vás neomezuje v tom, jaký konkrétní překladač použijete, ale nabízí konfigurační dialogy pouze pro některé z nich. V současné době jsou dostupné pluginy pro gcc, g++, f77, pghpf (Portland Group High Performance Fortran Compiler), pgf77 (Portland Group Fortran 77 Compiler), dcc (Borland Delphi Compiler) and fpc (Free Pascal Compiler).
