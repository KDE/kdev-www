Obsahuje všechny vlastnosti KDevelop 3.0.x plus:
  • Code Snippets (šablony zdrojového kódu)
    • SnippetPart přidává toolview, které je standardně umístěno v pravé dokovací liště
    • Přidávání, editace a mazání snippetů je umožněno přes kontextové menu
    • Dvojklikem na snippet v seznamu jej vložíte do právě aktivního pohledu na aktuální pozici kurzoru
    • Bublinové nápovědy zobrazují obsah snippetu
    • Snippety jsou uloženy v domovském adresáři uživatele, takže každý uživatel může mít své vlastní snippety
    • Snippety mohou obsahovat proměnné ve stylu $PROMENNA$. Při použití snippetu je uživatel dotázán na hodnotu, kterou bude proměnná nahrazena.
  • Integrace Doxygenu (Generátor API dokumentace)
    • Podporuje všechny programovací jazyky jako doxygen (kromě IDL a D)
    • Zobrazí náhled dokumentace, kterou vygeneruje doxygen pro soubor, který právě editujete
    • Konfigurace doxygenu
    • Spouštění doxygenu
    • Zobrazuje dokumenty vygenerované doxygenem (html man nebo pdf formáty)
    • Může použít XML soubory vygenerované doxygenem jako bublinovou nápovědu pro code completion
  • Prohlížeč dokumentace
    • Jednotné prostředí pro prohlížení dokumentace
    • Možnosti vyhledávání v obsahu, indexu nebo fulltextové vyhledávání (pomocí htdig).
    • Autodetekce a připojení dokumentace pro Qt, KDE Libs a DevHelp.
    • Podpora pro uživatelem volitelné soubory dokumentace (KDevelopTOC formát).
    • Schopnost otevřít lokální, vzdálené nebo komprimované soubory s dokumentací (používají se protokoly podporované Konquerorem - http, ftp, sftp, tar, zip, atd.).
    • Podporuje formáty TXT, HTML, DevHelp, CHM, PDF, Djvu a PDB.
    • Může použít jakýkoli formát dokumentace, který podporuje vaše instalace KDE
  • Flexibilní aplikační wizard (projektové šablony)
    • Aplikační wizard generuje kompletní, ke kompilaci připravené ukázkové programy.
    • Existují šablony pro každý programovací jazyk a sestavovací systém, který KDevelop podporuje.
    • Možnost použít vlastní šablony, které nejsou obsaženy v distribuci.
    • Obsahuje všechny šablony obsažené v KDevelop 3.0, plus šablony GTK, GNOME a Bonobo aplikací
    • Podporuje věci, o kterých ví pouze Ian :)
  • RAD (Rapid Application Development) orientace
    • Nový program kdevdesigner umožňuje nerušený vývoj Qt aplikací v QTDesigneru integrovaném v KDE.
    • Nový program kdevassistant umožňuje nápovědu v QT stylu.
    • Tyto programy mohou být použity samostatně, nebo z prostředí KDevelop, což vám umožní použít pouze jeden program pro vaše vývojářské potřeby.
