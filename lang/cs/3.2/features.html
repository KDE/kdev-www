Obsahuje všechny vlastnosti KDevelop 3.1.x plus:
  • Nově přepracovaný zdrojový kód
    • Zdrojový kód KDevelopu byl přepracován tak, aby se zlepšila znovuvyužitelnost kódu a zpřehlednila se dokumentace základních tříd knihoven, které jsou jádrem KDevelop. Tyto základní třídy tvoří novou platformu KDevelop a prezentují výstižné a dobře zdokumentované API
  • Nové RAD (Rapid Application Development) nástroje
    • Integrace KDevDesigner umožňuje design GUI přímo z prostředí KDevelop
    • Integrace KDevAssistant zpřístupnila bohatě vybavený prohlížeč dokumentace přímo v prostředí KDevelop
    • C++ Class Wizard může být nyní použit pro třídy v trvalém úložišti tříd (persistant class store). To znamená např. to, že pokud byla vytvořena databáze tříd Qt pro Code Completion, ve wizardu je možné vytvářet potomky tříd Qt
  • Zlepšená podpora jazyka ruby
    • Nový debugger ruby, kombinovaný s vylepšenými metodami pro prohlížení ve spojení qtruby/korundum, který zobrazuje Qt::Object metadata jako vlastnosti v průběhu ladění
    • Prohlížeč tříd má vylepšený parser, který zobrazuje Qt a DCOP signály a sloty, attr_accessor metody a proměnné instance
    • Nové šablony projektů pro KDE application framework, KConfig XT, DCOP server a aplikace psané pouze v Qt
    • Plná integrace Qt designeru/rbuic
  • Další vylepšení a opravy chyb
    • Byla předělána podpora správy verzí. Většina integračních vlastností, které byly v KDevelop 3.1 zakázány, jsou ve verzi 3.2 zpět, společně se značně vylepšeným pluginem pro Subversion.
    • Vylepšeno "Find in files", Ctags podpora a historie navigace zrychlují orientaci v kódu
    • Více šablon aplikací
    • nové šablony pro C/C++ aplikace a knihovny, založené na cmake
    • nové šablony pro kde aplikace založené na scons
    • více informací najdete na stránce změn
