Має зміт можливостей KDevelop 3.2.x plus:
  • Нові можливості у секції KDevelop на KDE 3.5 Feature Plan
  • Як використовувати CMAKE для компіляції KDevelop
  • Improved C++ language support:
    • підтримка QT4, інший код генерується для Qt3.x та Qt4.x проектів
    • Added new tab у діалог опцій C++ для конфігурації використання Qt бібліотеки. Ви можете конфігурувати, якщо Qt використовується, which major version і де бібліотека Qt інстальвана.
  • Improved debugger для мов C, C++ та Ruby:
    • Debugger breakpoints window made easier to use
    • Автоматично показує вікно з змінними коли програма stop, і скриває, коли програма закінчується .
    • Implement editing expression/variable values in variables window.
    • Improved the variables view
  • PHP5 підтримує
  • Новий менеджер проєктівr:
    • Він базується на архітектурі платформи kdevelop де GUI проекту абстрактний від системи. Система використовує плагіни("project importers, builders and editors")
    • Підтримка Workspace
    • Project builders: make builder
    • Project importers: automake, custom makefiles, qmake
  • Дивись сторінку changelog для деталей
