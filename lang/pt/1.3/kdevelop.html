Esta página concentra toda a informação sobre a série KDevelop 1.x

O desenvolvimento no KDE 1.x terminou em 11 de Dezembro de 2000 com o lançamento do KDevelop 1.3, e não se encontram planeadas novas versões do KDevelop 1.x para KDE 1.x.
O KDevelop 1.4 destina-se ao KDE 2.X e Qt 2.x. mas use antes o KDevelop 2.1.5 se necessita do KDevelop em KDE 2.x.
Actualmente apenas restam quatro desejos para o KDevelop 3.0 (para KDE 3.0, Qt 3.0) contrariamente aos muitos bugs e desejos do KDevelop 1.3.

Plugins e correcções

Devido a várias razões (principalmente a estabilidade), algumas correcções maiores não podem ser aplicadas na versão currente. Estas no entanto oferecem grandes melhoramentos, por isso podem ser descarregadas aqui.
Lembre-se que se tratam de correcções para o código, logo terá que descarregar uma versão do código compatível, aplicar as correções e recompilar.

Compilar a versão 1.3 a partir do repositório CVS

Para obter automaticamente o KDevelop compilado a partir das fontes em cvs, faça o checkout do módulo CVS "kdevelop" em :pserver:anonymous@anoncvs.kde.org:/home/kde utilizando o ramo CVS corrigido KDEVELOP_1_0

linha de comandos se estiver a correr KDE 1.x:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Por forma a compilar e instalar o KDevelop (versão CVS) no seu sistema, escreva os comandos abaixo na directoria base da distribuição do KDevelop:

para uma versão de lançamento: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

para uma versão de debug: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)

Bugs conhecidos

Devido ao desenvolvimento desta série ter terminado, existem vários bugs conhecidos mas não corrigidos. As boas notícias são que todos esses bugs se encontram corrigidos no KDevelop 3.0. Por isso considere antes por favor a utilização da versão 3.0.
