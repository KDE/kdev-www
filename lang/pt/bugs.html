Para reportar um bug ou uma sugestão para uma nova funcionalidade, visite o assistente para relatório de bugs da nossa base de dados bugzilla. Não deve reportar erros de compilação na base de dados de bugs. Para esse efeito utilize antes a lista de discussão.

Para obter a lista completa dos correntes bugs e sugestões pesquise na base de dados de bugs. Os relatórios estão organizados por componentes, permitindo obter uma listagem por componente e sua gravidade.

Para obter uma descrição detalhada dos bugs, active os cookies no seu browser e vá de seguida á página de configuração do bugzilla e seleccione:
  • changeddate
  • severity
  • priority
  • reporter
  • status
  • component
  • version
  • votes
  • Summary (first 60 caracters)

E é tudo, a partir de agora todas as suas consultas de bugs irão apresentar todos os detalhes que precisa para o KDevelop.
