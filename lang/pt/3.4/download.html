Binários :

Alguns distribuidores de sistemas Linux/UNIX doaram generosamente pacotes binários do KDevelop para algumas versões das suas distribuições, e noutros casos voluntários da comunidade KDE fizeram-no. Alguns desses pacotes binários estão disponíveis gratuitamente por FTP ou http em mirrors KDE.



Código fonte para KDE >= :


Ou pode compilar o código fonte obtido através do repositório SVN do KDevelop.

  1. Antes de continuar verifique por favor se o seu computador tem o software necessário instalado.

  2. Descomprima o ficheiro fonte e mude para a directoria base:
    $ bunzip2 kdevelop-.tar.bz2
    $ tar -xf kdevelop-.tar
    $ cd kdevelop-
