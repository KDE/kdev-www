Antes de iniciar a compilação da versão que acabou de descarregar do KDevelop, tenha a certeza que tem instalados os seguintes programas. Lembre-se que pode obter erros estranhos, se utilizar versões mais antigas.
Normalmennte pode conhecer a versão do software instalada no seu computador executando o programa com o argumento "--version". Por exemplo, para descobrir qual é o autoconf que tem instalado na sua máquina faça: 
   autoconf --version

Estas ferramentas são opcionais, o que significa que o KDevelop funciona sem elas. No entanto, não estarão disponiveis algumas funcionalidades como geração de documentação ou pesquisas sobre vários ficheiros. Logo, provavelmente deverá verificar no seu sistema a existência destes programas.

Note que em ordem a utilizar o suporte do KDevelop para linguagens como ada, pascal, java, etc. precisa do compilador correspondente. O KDevelop não limita a escolha do compilador a utilizar mas oferece diálogos de opções de compilação para alguns deles. Actualmente os plugins de compilação existentes são para gcc, g++, f77, pghpf (Portland Group High Performance Fortran Compiler), pgf77 (Portland Group Fortran 77 Compiler), dcc (Borland Delphi Compiler) e fpc (Free Pascal Compiler).
