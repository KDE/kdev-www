Contém todas as características do KDevelop 3.2.x mais:
  • As novas características da secção do KDevelop no plano de características do KDE 3.5
  • Agora pode-se usar o CMAKE para compilar o KDevelop
  • Suporte da linguagem C++ melhorado:
    • Suporte ao QT4, geração de código diferente para projectos Qt3.x e Qt4.x
    • Acrescentado novo item no diálogo de opções C++ para configurar biblioteca Qt usada. Se usar o Qt, pode configurar que versão e onde a biblioteca Qt está instalada.
  • Depurador melhorado para as linguagens C, C++ e Ruby:
    • Janela de 'breakpoints' do depurador mais fácil de usar
    • Mostra automáticamente a janela de variáveis quando a aplicação pára, e esconde-a quando a aplicação termina.
    • Edição de valores expressão/variável na janela variáveis.
    • Melhoria da vista de variáveis
  • Suporte à linguagem PHP5
  • Novo gerenciador genérico de projectos:
    • Baseado na arquitetura da plataforma KDevelop onde o gerenciamento de projectos da GUI é abstraído de um sistema concreto. Um dado sistema é manipulado usando-se plugins ("importadores de projetos, construtores e editores")
    • Suporte à área de trabalho disponível
    • Constructores de projetos: constructor do make
    • Importadores de projetos: automake, makefiles cliente, qmake
  • Veja a página change log para maiores detalhes
