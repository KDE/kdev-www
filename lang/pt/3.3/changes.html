Suporta o KDE >= . Foi marcado em 2005-07-26 mas não foi lançado. Contém as novas funções do KDevelop mais as alterações:
  • NOVO - Funções na secção do KDevelop KDE 3.5 Feature Plan
  • NOVO - Agora pode-se usar o CMAKE para compilar o KDevelop
  • MELHORADO - suporte à lingua Cpp:
    • suporte para QT4
    • Adicionada nova aba nas opções de C++ para configurar a biblioteca Qt utilizada. Pode-se configurar se Qt é utilizado, qual a versão e onde está a biblioteca Qt (QTDIR) para este projecto.
    • assistente class: - as caixas de verificação Qt estão desactivadas em projectos não-Qt. Código diferente é gerado para projectos Qt3.x e Qt4.x.
  • MELHORADO - Depurador:
    • Janela de pontos de bloqueio agora mais fácil de utilizar
    • Corrigido problema compontos de bloqueio desactivados
    • Automaticamente exibir a janela de variaveis quando o programa pára, e esconder quando o programa termina.
    • Implementado edição de valores de expressões/variáveis na janela de variáveis.
    • Melhroada a janela de variáveis
  • NOVO - Suporte para lingua PHP5, confiurar localização executavel php
  • MELHORADO - Janela de saída:
    • Optimizações de velocidade
  • MELHORADO - Gestor de projectos geral:
    • É baseado na arquitectura da plataforma kdevelop onde a interface gráfica da gestão de projectos é abstrata de um sistema de contrução especifico. Sistemas de contrução são manipulados utilizando plugins ("Importadores de projectos, construtores e editores")
    • Suporte a Área de Trabalho está disponível
    • Construtor de projecto: make builder
    • Importador de projectos: automake, custom makefiles, qmake
  • MELHORADO - Manual utilizador
  • CORRIGIDO - cerca de 48 erros da série KDevelop 3.2.x
