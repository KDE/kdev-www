Esta página concentra toda a informação sobre a série KDevelop 3.0.x

Development started under the Gideon codename. The email A new IDE for a new milleneum :-) from Bernd Gehrmann describes how it all started.
KDevelop 3.0 is a stable release for KDE-3.x, with a entirely new code base, many new features (e.g. support of new programming languages, new editors). A Versão 3.0.0 foi lançada juntamente com o KDE-3.2.0. Versões bugfix do KDevelop 3.0.x serão lançadas juntamente com o KDE-3.2.x
