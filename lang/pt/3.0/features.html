• Arquitectura baseada em plugins
    • Novas funcionalidades pode ser adicionadas, substituídas ou removidas sem alterar o código fonte principal.
    • Licença de bibliotecas (LGPL) permite a criação de plugins para o IDE comerciais e grátis IDE.
    • Suporte para perfis que determinam um conjunto de plugins a serem carregados. Isto é especialmente útil para ambientes restritos ou configurações personalizadas.
  • Gestão de Projectos
    • Suporte nativo não intrusivo para automake e qmake. Modificações manuais são permitidas e reconhecidas pelo IDE, projectos e sub-projectos existentes podem ser facilmente importados.
    • Habilidade para usar makefiles customizados para construir o projecto.
    • Suporte para ant - ferramenta de construção para projectos java.
    • Suporte para tipos de projecto extra como pascal, haskell, ada e projectos de script.
    • Uma estrutura para adicionar suporte para qualquer sistema de construção criando plugins de construção para o gestor de projectos genéricos.
    • Assistente de Novo ficheiro para gerir modelos de ficheiro e criar novos ficheiros no projecto a partir de modelos.
    • Plugins de compilador para definirem opções de compilação em diálogos convenientes.
    • Perfis de configuração com a habilidade de especificar diferentes compiladores, flags de compilação, directorias de construção, arquitectura alvo, etc.
    • Funcionalidades para documentação do projecto tais como suporte para a criação automática de documentação API utilizando doxygen (em formato html, tex ou man) com referências cruzadas para as bibliotecas usadas.
    • Suporte para internacionalização, permitindo aos tradutores adicionarem uma linguagem facilmente ao projecto, incluindo suporte para KBabel.
    • Suporte para projectos Qt/Embedded (para as plataformas Zaurus e iPAQ).
  • Suporte de Linguagens
    • 15 linguagens suportadas incluindo Ada, C, C++, Objective-C (atráves de suporte C), SQL, Fortran, Haskell, Java, PHP, Pascal, Perl, Python, Ruby, Bash, XUL (não oficial).
    • Relatório de problemas que anuncia vários "problemas" com o código fonte á medida que se escreve. Estes problemas incluem erros de sintaxe, TODO's e FIXME's.
    • Explorador de classes que mostra a hierarquia de classes e outros símbolos no projecto.
    • Parsers de linguagens utilizados pelo explorador de classes e relatório de problemas.
    • Assistente de classes, métodos e atributos para linguagens orientadas por objectos.
    • Completação automática de código e dicas de código para variáveis de classe, métodos, argumentos de funções e outros.
    • Formatação de código, highlight sintáctico e "dobragem" de código.
    • Funcionalidades especificas de C++, tais como
      • "criar membro" - criação da definição de métodos no ficheiro fonte baseado na declaração do método no ficheiro de cabeçalho;
      • "extrair interface"- extracção de interface (métodos declarados) e criação de classes bases abstractas com essa interface;
      • "sub-classes" - criação automática de sub-classes e implementação de slots para formulários QT Designer.
  • Debugger Integrado
    • Frontends para GDB e debugger Java.
    • Consola GDB integrada.
    • Watchpoints e breakpoints em linhas, endereços e métodos.
    • Breakpoints condicionais.
    • Desassemblador e visualizador da pilha de chamadas.
    • Árvore de visualização de variáveis.
    • Visualizadores de memória, registos e bibliotecas.
    • Suporte para shell de debug (libtool).
    • Debugging remoto.
  • Edição de código
    • Qualquer editor que implemente a interface KDE KTextEditor pode ser utilizado com o KDevelop.
    • Actualmente os editores disponíveis são o Kate, QEditor (baseado no editor do Qt Designer) e KVim.
    • Marcadores abrangendo todo o projecto que são armazenados no ficheiro de projecto.
    • Visualizador de diferenças entre o texto em edição e o texto no disco.
    • Visualizador de diferenças para ficheiros de correcção.
    • Integração com o Grep - pesquisas no projecto utilizando expressões regulares.
    • Pesquisa e substituição abrangendo todo o projecto.
  • Sistema de Controlo de Versões
    • Uma API comum para incluir suporte para controlo de versões no KDevelop.
    • Sistemas de controlo de versões suportados: CVS, Perforce, Subversion e Clearcase.
  • Assistente de Aplicação
    • O assistente de aplicação gera aplicações de exemplo completas, prontas a serem utilizadas.
    • Existem modelos para cada uma linguagens suportadas e diferentes sistemas de construção.
    • Possibilidade de utilizar modelos de aplicação personalizados não incluídos na distribuição.
    • Um punhado de modelos de projectos para KDE (aplicações KPart,aplicações KDE simples, módulos KControl, applets Kicker (painel), parts KOffice, KIOSlaves, plugins Konqueror, Noatun e Kate e estilos de ambiente de trabalho).
  • Visualização de Documentação
    • Uma interface unificada para exploração da documentação.
    • Capacidade para tabela de conteúdos, indicies e pesquisa total de texto (usando htdig).
    • Autodetecção e inclusão de documentação Qt, KDE Libs e DevHelp.
    • Suporte para colecções de documentação costumizada (formato KDevelopTOC).
    • Capacidade para abrir documentação local, remota ou mesmo comprimida (utilizando qualquer protocolo suportado pelo Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Capacidade para abrir documentação em vários formatos (html, text, ps, pdf, chm, etc.).
  • Outras Características
    • A interface de utilizador é completamente configurável, permitindo a mudança no momento do modo entre os modos clássicos MDI como o toplevel, janela filha e o novo modo IDEAl que disponibiliza barras de ferramentas deslizantes.
    • Consola embebida.
    • Selector de ficheiros, árvore de ficheiros e grupo de ficheiros, funcionalidades que eliminam a necessidade de um gestor de ficheiros externo.
    • Inclusão de um qualquer programa que necessite para desenvolvimento pela sua adição ao menu de "Ferramentas" de acordo com as suas necessidades particulares.
    • Testador de expressões regulares.
    • Integração do Valgrind permite a detecção de fugas de memória.
    • Ferramenta para distribuição e empacotamento do projecto para criação automática de pacotes de código fonte e binários.
    • Integração CTags.
    • Explorador de Part's para pesquisar os diversos serviços KDE.
