Código fonte para KDE >= :

Versão de desenvolvimento atual, e como tal instável, atualizada cada 24H (tar.bz2)

  1. Antes de continuar verifique por favor se o seu computador possui o software necessário instalado.

  2. Descarregue o ficheiro fonte tar.bz2.

  3. Descomprima o arquivo fonte e mude para o directorio base:
    $ bunzip2 kdevelop.tar.bz2
    $ tar -xf kdevelop.tar
    $ cd kdevelop
