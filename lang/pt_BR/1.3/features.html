Gerenciamento de Projetos:

O arquivo de projeto mantem toda informação para os arquivos do projeto como propriedades de arquivo (inclui ou exclui da distribuição), permitindo a criação e alteração de projetos individualmente. Os projetos gerados são compatíveis com autoconf/automake.

Editor de Diálogos :

O KDevelop oferece um meio fácil de criar GUI (Graphical User Interface - Interface Gráfica para Usuário) com o Editor de Diálogos interno. Você pode deixar que o KDevelop gere o código-fonte do diálogo e ter controle total de suas funcionalidades.

Classparser / Classtools:

O visualizador de classes atualmente analisa gramaticalmente quase toda instrução C++ e C, classes aninhadas, estruturas dentro de classes e operadores, assim como namespaces. Também é possível adicionar métodos e atributos usando os diálogos das ferramentas de classes. As ferramentas de classe (classtool) são novos recursos do KDevelop. A idéia e que você geralmente deseja uma ferramenta que mostre a herança e os métodos que são utilizáveis através desta herança. Esta necessidade é preenchida com a ferramenta de classes (classtool). Trata-se de um meio intuitivo para navegar classes no sistema e ver seus relacionamentos.

Debugador Integrado

O debugador interno que acompanha o KDevelop 1.1, permite que a utilização mais eficiente do visualizador de classes durante uma sessão de debuger, facilitando o acesso ao código-fonte para colocar pontos-de-parada e visualizar o conteúdo das variáveis da sua aplicação.

Visualizador de Classes Gráfico:

O Visualizador de Classes Gráficos oferece a possibilidade de ter uma visão geral do projeto e de todas as suas classes.

Assistente de Aplicações:

O KAppWizard permite a geração de diferentes frameworks de aplicações para servirem de ponto-de-partida para novos programas. Ele pode gerar :

  • Uma aplicação KDE padrão com barra de menus, barra de ferramentas e barra de status;
  • Uma mini aplicação KDE com uma janela principal vazia;
  • Uma aplicação GNOME completa;
  • Uma aplicação baseada exclusivamente em QT, tambem com barra de menus, barra de ferramentas e barra de status;
  • Aplicações tipo terminal C/C++. O KDevelop 1.1 inclui frameworks KDE 2/QT 2 para aqueles desenvolvedores que querem permanecer no topo do desenvolvimento KDE. Tudo isto é gerado e adaptado às necessidades da sua nova aplicação e estará pronto para "rodar" em segundos.

    Documentação:

    O KDevelop vem com um conjunto completo de documentação que consiste de cinco manuais:

  • O manual do usuário;
  • O manual de programação;
  • O tutorial;
  • Guia de referência da biblioteca KDE.

    Esta documentação mais a referêcia do C/C++ e a documentação do QT e do KDE podem ser facilmente acessadas pelo browser de documentação integrado.

    Quatro Treeviews:

    As treeviews permitem a fácil troca entre arquivos fonte/header, classes e documentação que o habilita a gerenciar seus arquivos do projeto sem usar um gerenciador de arquivos externo.

    Editor Integrado:

    Editor completo, incluindo realce de sintaxe. (KWrite)

    Ferramentas de Desenvolvimento Unix integradas:

    Nós integramos um punhado de recursos que podem ser muito úteis. Também durante os testes do KDevelop, tivemos novas idéias para facilitar a rotina de trabalho e tentamos também implementá-las. Estamos falando de : kdoc, kiconedit, kdbg, automake & friends etc.

    Browser de Documentação Integrado:

    O Browser de documentação oferece acesso direto à documentação atual das bibliotecas do QT e do KDE; aos manuais e tutoriais do KDevelop. Com a ajuda da árvore de documentação você pode navegar diretamente para o tópico de qualquer classe ou método KDE/Qt.

    Poderos sistema de Ajuda:

    O sistema de ajuda on-line (helpsystem) possui mensagens 'QuickHelp', tipo "What's this - O que é isto" - e mensagens de barra de status em qualquer lugar. Desta forma, você sempre tem ajuda sobre onde você está e o que pode fazer.
