• Arquitetura baseada em Plugins
    • Functionalidades podem ser adicionadas, substituídas e removidas sem alterar o código fonte central.
    • A licença das bibliotecas (LGPL) permite criar plugins da IDE comerciais e livres.
    • Suporte a perfis que determinam qual conjunto de plugins serão carregados. Isto é especialmente útil para ambiente restristos ou configurações personalizadas..
  • Gerenciamento de Projetos
    • Suporte não intrusivo nativo ao automake e o qmake. Modificações feitas à mão são permitidas pela IDE, além de projetos e subprojetos serem facilmente importados.
    • Habidade em usar makefiles personalizados para construir o projeto.
    • Apoio a ferramenta de construção ant - para projetos em java.
    • Apoio a outros tipos de projetos como para pascal, haskell, ada e projetos de scripts.
    • Um framework para adicionar suporte a qualquer sistema de construção com a criação de plugins para o gerenciador genérico de projetos.
    • Novo assistente para gerenciar os modelos de arquivos e criar novos arquivos no projeto a partir dos modelos.
    • Plugins de compiladores para definir as opções de compilação em diálogos convenientes.
    • Perfis de configuração com a habilidade de especificar diferentes compiladores, flags de compilação, diretórios de construção, arquitetura de alvos, etc.
    • Facilidades para documentação de projetos como suporte ao doxygen na criação automica de documentação de API (nos formatos html, tex e man) para projetos com referências cruzadas às bibliotecas utilizdas.
    • Suporte a internationalizaçãp, permitindo os tradutores adicionarem a sua linguagem padrão ao projeto facilmente, incluindo suporte ao KBabel.
    • Suporte a projetos Qt/Embedded (para as plataformas Zaurus e iPAQ platforms).
  • Suporte a Linguagens
    • 15 linguagens suportadas incluindo Ada, C, C++, Objective-C (via apoio C), SQL, Fortran, Haskell, Java, PHP, Pascal, Perl, Python, Ruby, Bash, XUL (não oficialmente).
    • Relator de Problemas - que relata varios "ploblemas" com o código fonte no momento da digitação. Erros incluindo erros de sintaxe da linguagem, afazeres (TODO) e "conserte-me" (FIXME).
    • Navegador de classes que mostra a hierarquia das classes e outros símbolos no projeto.
    • Interpretador de linguagem que alimenta o navegador de classes e o relator de problemas.
    • Assistentes de classe, método e atributos para linguagens orientadas a objeto.
    • Completação de código automática e dicas de código para variáveis de classe, métodos, argumentos de funções e mais.
    • Formatação do código-fonte, destaque de sintaxe e "code folding".
    • Funcionalidades específicas de C++, como
      • "make member" - cria definição de métodos no fonte baseado na declaração do cabeçalho;
      • "extract interface"- extrai a interface (declaração de métodos) e cria uma classe base abstrata com esta interface;
      • "subclassing" - criação de subclasses automatizada com implementação de "slots" para formulários do Qt Designer.
  • Depurador Integrado
    • Front-end para o GDB e o depurador Java.
    • Console integrado com o GDB.
    • Breakpoits e watchpoints para linha, endereço e métodos.
    • Breakpoints condicionais.
    • Disassembler e visualizador do quadro de pilha.
    • Árvore de acompanhamento de variáveis.
    • Visualizadores de memória, registradores e bibliotecas.
    • Suporte a shell de depuração (libtool).
    • Depuração remota.
  • Ediçãp de Código
    • Qualquer editor que implemente a interface do KDE KTextEditor, pode ser utilizado com o KDevelop.
    • Atualmente os editores disponíveis são Kate, QEditor (baseado no editor do Qt Designer) e KVim.
    • Bookmarks com abrangência de projeto que são armazenados no arquivo de projectos.
    • Visualizador de diferenças entre o texto editado no momento e o texto no disco.
    • Visualizador de diferenças para os arquivos de "patch".
    • Integração com o Grep - busca com expressões regulares.
    • Ferramenta de pesquisa e substituição com abrangência de projeto.
  • Sistemas de Controle de Versões
    • Uma API comum para incluir apoio a SCV no KDevelop.
    • Sistemas SCV suportadoss: CVS, Perforce, Subversion e Clearcase.
  • Assistente de Aplicações
    • Assistente de Aplicações geram aplicações amostrais completas e prontas para usar.
    • Modelos para cada linguagem suportada e sistema de construção existentes.
    • Habilidade de utilizar modelos de aplicação personalizados não incluídos na distribuição.
    • Muitos modelos de projetos relacionados ao KDE (aplicações KPart, aplicações KDE simples, módulos KControl, applets Kicker(panel), partes do KOffice, KIOSlaves, Konqueror, plugins Noatun e Kate e Estilos de Desktop).
  • Visualizador de Documentação
    • Uma interface unificada para navegação na documentação.
    • Sumário, índice e pesquisa integral no texto (com htdig).
    • Auto-detecção e inclusão de documentação de bibliotecas Qt, KDE e DevHelp.
    • Apoio a coleções de documentação personalizadas (formato KDevelopTOC).
    • Habilidade de abrir documentação local, remota e até comprimida (usando qualquer protocolo suportado pelo Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Habilidade de abrir vários formatos de documentação (html, text, ps, pdf, chm, etc.).
  • Outras Funcionalidades
    • A interface com o usuário é completamente configurável, o que permite mudanças "on the fly" do modo clássico MDI entre o toplevel, childframe e o novo modo IDEAl que apresenta visões da ferramenta em janelas laterais deslizantes.
    • Console imbutido.
    • Seletor de arquivos, árvore de arquivos e grupos de arquivos File selector, facilidades que tornam desnecessário a utilização de um gerenciador externo de arquivos.
    • A inclusão de qualquer outro programa que você precise para desenvolver, adicionando-o ao menu "Ferramentas" de acordo com suas necessidades individuais.
    • Testador de expressões regulares.
    • Integração com o Valgrind para encrontrar vazamentos de memória no programa.
    • Ferramenta de distribuição e empacotamento de projetos para a criação automática de pacotes binários e fontes.
    • Integração com CTags.
    • Explorador de "Parts" para consultar serviços do KDE disponíveis.
