PARA LANÇAMENTO IMEDIATO

Anúncio de lançamento do KDevelop 3.0

A Equipe do KDevelop faz o Lançamento de Importantes Funcionalidades do Premiado IDE de Código Aberto

03 de Fevereiro de 2004 (A INTERNET). A Equipe do KDevelop está orgulhosa em apresentar KDevelop 3.0, o primeiro grande lançamento do completamente reestcrito, premiado Ambiente de Desenvolvimento Integrado (IDE).

"O KDevelop é a chave para se alcançar novos níveis de produtividade. Ele é um IDE apropriado para todas as tarefas desde programação de sistemas até desenvolvimento rápido de interfaces para usuário (GUI RAD). Integrando todas as ferramentas convencionais para desenvolvimento no UNIX, provendo suporte a linguagens, KDevelop é uma excelente escolha para desenvolvedores UNIX. Disponível para diferentes plataformas como Mac OS X e Windows/Cygwin, o KDevelop é uma grande ferramenta para desenvolvimento multi-plataforma. Apoio nas ferramentas comuns de desenvolvimento permite desvolver aplicações para um vasto número de plataformas incluindo plataformas embutidas" disse Alexander Dymo, co-mantenedor do projeto.

"Com um núcleo independente de linguagens, baseado em plugins e com vasto suporte às ferramentas comuns de desenvolvimento, o KDevelop já é de fato o IDE padrão para desenvolvimento para desktops Linux/Unix" acrescentou Harald Fernengel, membro antigo do projeto.

"A equipe do KDevelop fez um excelente trabalho com este lançamento. Mais de 40 desenvolvedores contribuíram com o projeto num espaço de tempo de 14 meses. Você notará que foi tomado muito cuidado com a documentação e com a interface com o usuário. Esta vesão é poderosa, além de ser fácil de usar" observou Amilcar do Carmo Lucas, co-coordenador de lançamento.

Grandes Melhorias

KDevelop agora dá suporte a mais de 15 linguagens de programação e vários sistemas de controle de versão, debuggers, formatos de documentação e ferramentas de compilação. Qualquer editor que suporta as interfaces do KTextEditor pode ser utilizado como editor nativo do KDevelop incluindo kate, QEditor e kvim. Ferramentas de compilação como automake, qmake ou ant são suportadas nativamente, o que significa que o KDevelop não não seu próprio repositório de objetos. Modificações nos arquivos nativos do projeto irão refletir no KDevelop e vice-versa.

O suporte a C++ oferece uma visão das classes, mostrando todos os símbolbos de forma simples ou hierarquica por namespaces e permite fácil navegação pelo código. Um interpretador em background atualiza os símbolos automaticamente e também mostra erros sintáticos no código-fonte. O recurso de code completion também lidá com os recursos de signals e slots do toolkit Qt.

Com sua nova arquitetura aberta, o KDevelop pode ser personalizado em vários aspectos. A interface com o usuário pode ser chaveada a qualquer momento do clássico modo MDI para o moderno modo IDEAl, oferecendo uma abordagem centrada no código com visões das ferramentas por demanda.

O KDevelop oferece um suporte estensivo para navegação no código-fonte. Símbolos e nomes de arquivos podem ser acessados apenas digitando parte do nome. Arquivos modificados recentemente podem ser recuperados com uma simples combinação de teclas.

O KDevelop oferece geradores de documentação, depuração de código, verificação de memória, reengenharia de código, gerenciamento de bookmarks, e dúzias de ferramentas que tornam o desenvolvimento fácil e conveniente. Além disto, ele é traduzido em mais de 16 idiomas do Inglês ao Tamil.

Instalando os pacotes binários do KDevelop 3.0

Pacotes binários para todas as grandes distribuições do Linux podem ser encontrados em http://www.kdevelop.org/index.html?filename=3.0/download.html.

Uma versão alfa do binário nativo para o Mac OS X pode ser encontrado em http://kde.opendarwin.org.

O projeto kde-cygwin provê o KDevelop em http://kde-cygwin.sourceforge.net/kde3/installation.php.

Favor observar que a equipe do KDevelop não fornece os binários - todos os pacotes são disponibilizados por terceiros.

Compilando o KDevelop 3.0

O código-fonte completo do KDevelop 3.0 pode ser baixado gratuitamente no site http://www.kdevelop.org. Informações detalhadas para compilação podem ser encontradas em http://www.kdevelop.org/index.html?filename=3.0/branches_compiling.html.

Sobre o KDevelop

O KDevelop é um projeto independente, para se criar um ambiente de desenvolvimento integrado (IDE) poderoso e de fácil utilização. Como parte do projeto KDE, KDevelop compartilha o mesmo modelo de desenvolvimento e estrutura modular compartilhada. O resultado é um IDE altamente personalizável, independente de linguagem de programação e licenciado sobre a GPL por dúzias de desenvolvedores e profissionais espalhados pelo mundo. Para saber mais sobre o KDevelop veja http://www.kdevelop.org.

Sobre o KDE

KDE é um projeto independente com centenas de desenvolvedores, tradutores, artistas e outros profissionais espalhados pelo mundo colaborando pela Interner para se criar e distribuir gratuitamente um sofisticado, personalizável e estável ambiente para desktop e escritórios empregando uma arquitetura flexível, baseada em componentes, transparente a rede e oferecendo uma excepcional plataforma para desenvolvimento. O KDE provê um estável e maduro desktop, uma suite de escritório (KOffice) completo e baseado em coponentes, um vasto conjunto de ferramentas e utilitários administrativos e de rede e um eficiente e intuitivo ambiente de desenvolvimento com o IDE KDevelop. KDE é uma prova funcional de um software de Código Aberto do "Estilo Bazar" de desenvolvimento, que pode se comparar ou até superar tecnologias de ponta dos sistemas comerciais mais complexos.
