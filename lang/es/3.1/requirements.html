Antes de empezar a compilar tu reciente versión de KDevelop, asegúrate que tienes instalado los siguientes programas. Recuerda que puedes tener errores extraños, si usas versiones antiguas.
Normalmente puedes ver que versión del software esta instalado en tu computador ejecutando el programa con el argumento "--version". Por ejemplo para buscar que autoconf tienes instalado en tu máquina haz: 
   autoconf --version

Programa/Biblioteca Fuente
KDE >= 3.2.0 www.kde.org
Qt >= 3.2.3 www.trolltech.com/products/qt
g++ >= 2.95.3 (o compatible) gcc.gnu.org
GNU make www.gnu.org/software/make
perl >= 5.004 www.perl.com
autoconf >= 2.52 www.gnu.org/software/autoconf
automake >= 1.6 www.gnu.org/software/automake
flex >= 2.5.4 www.gnu.org/software/flex


Estas herramientas son opcionales, esto significa que KDevelop funcionará sin ellas. Pero carecerá de algunas funcionalidades como generar documentación o búsquedas sobre varios ficheros. Así que probablemente deberá comprobar si en su sistema están estos programas.

Por favor dese cuenta que para usar lenguajes de programación soportados como ada, pascal, java, etc. necesitará el correspondiente compilador del lenguaje. KDevelop no limita el compilador a usar pero ofrece diálogos de opciones de compilación para algunos de ellos. Actualmente se disponen de los siguientes plugins de compilación gcc, g++, f77, pghpf (Portland Group High Performance Fortran Compiler), pgf77 (Portland Group Fortran 77 Compiler), dcc (Borland Delphi Compiler) y fpc (Free Pascal Compiler).
