Contiene todas las funcionalidades de KDevelop 3.0.x más:
  • Recortes de Código
    • Los recortes de código añaden una herramienta visual que por defecto se ancla a al derecha
    • Se puede añadir, editar y borrar recortes a través de menús desplegables
    • Haciendo un doble clic un formulario de recortes la lista lo inserta dentro de la vista activa en la posición actual del cursor
    • La herramienta de sugerencias muestra el contenido del recorte
    • Los recortes se almacenan el directorio home de los usuarios, así cada usuario puede tener sus propios recortes
    • Los recortes pueden contener variables del estilo $VARNAME$. Al usar el recorte se le pide al usuario que introduzca el valor a reemplazar por las variables
  • Integración con Doxygen (generador de la documentación del API)
    • Soporte todos los lenguajes de programación que doxygen soporta (excepto IDL y D)
    • Previsualiza la documentación que genera doxygen para el fichero que estás editando actualmente
    • Configura doxygen
    • Ejecuta doxygen
    • Muestra los doc generados por doxygen (formatos html, man o pdf)
    • Puede usar los XML generados por doxygen como herramienta de sugerencias para la completación de código "a tu gusto"
  • Visor de documentación basado en plugin
    • Un interfaz unificado para la navegación de documentación.
    • capacidad de búsqueda de TOC, index y text completo (con htdig).
    • Autodetección e inclusión de documentación de Qt, Bibliotecas KDE y DevHelp.
    • Suporte para colecciones de documentación propias (formato KDevelopTOC).
    • Capacidad de abrir de forma local, remota e incluso documentación comprimida (usando cualquier protocolo soportado por Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Suporte de formatos de fichero TXT, HTML, DevHelp, CHM, PDF, Djvu y PDB.
    • Puede usar cualquier formato de documentación que soporte su KDE
  • Asistente de aplicaciones flexible (plantillas de proyecto)
    • Asistente de aplicaciones genera aplicaciones completas de ejemplo listas para funcionar.
    • Existe plantillas para cada uno de los lenguajes de soportados y para cada sistema de construcción.
    • Capacidad para usar plantillas de aplicaciones propias que no incluidas en la distribución.
    • Contiene todas las plantillas de KDevelop 3.0 más las plantillas de aplicaciones de GTK, GNOME y Bonobo
    • Suporte de cosas que sólo Ian conoce :)
  • Orientado a RAD (Desarrollo Rápido de Aplicaciones)
    • La nueva aplicación kdevdesigner proporciona un desarrollo integral de GUI basadas en QT a través de la integración de QTdesigner en el marco de trabajo de KDE.
    • La nueva aplicación kdevassistant proporciona documentación de ayuda al estilo QT.
    • Estas aplicaciones pueden ser usadas de forma independiente o desde dentro del propio KDevelop, esto proporciona un programa de desarrollo para todo lo que usted necesite.
