Binarios :

Algunos vendedores de Linux/ UNIX OS que amablemente proporcionan paquetes binarios de Kdevelop para muchas versiones de sus distribuciones, y en otros casos voluntarios de la comunidad también lo hacen. Algunos de estos paquetes binarios están disponibles para ser bajados libremente desde mirrors http o FTP de KDE.



Fuentes para KDE >= 3.0 :


Antes de continuar por favor comprueba si su computadora tiene instalado el software requerido.

Descomprima el fichero fuente y cambie el directorio base: 
       $ bunzip2 kdevelop-3.0.x.tar.bz2
       $ tar -xf kdevelop-3.0.x.tar
       $ cd kdevelop-3.0.x
