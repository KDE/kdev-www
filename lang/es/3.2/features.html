Contiene todas las funcionalidades de KDevelop 3.1.x más:
  • Código base rediseñado
    • El código base de KDevelop ha sido rediseñado con el objetivo de mejorar la reutilización de código y documentar la biblioteca de clases base que compone KDevelop. Las clases base forman la nueva plataforma KDevelop y presenta una concisa y bien documentada API
  • Nuevas herramientas RAD (Rapid Application Development)
    • La integración con KDevDesigner permite el diseño GUI desde dentro del propio KDevelop
    • La integración con KDevAssistent provee un visor de documentación rico en funcionalidades desde dentro del propio KDevelop
    • El asistente de clases C++ puede ahora ser usado para clases en el almacén permanente de clases. Esto significa, por ejemplo, que si la base de datos de Completado de Código para Qt está construída, entonces derivar de clases de Qt es posible en el propio asistente
  • Soporte del lenguaje Ruby muy mejorado
    • Nuevo depurador de Ruby, combinado con métodos mejorados de inspección en los nexos qtruby/korundum, para mostrar los metadatos Qt::Objet como propiedades, mientras se depura.
    • El navegador de clases tiene un analizador mejorado que muestra señales de Qt y DCOP y ranuras, métodos de acceso a atributos y variables instanciadas
    • Nuevas plantillas de proyecto para aplicación KDE, KConfig XT, servidor DCOP y aplicaciones sólo Qt
    • Integración completa con Qt designer/rbuic
  • Otras mejoras y arreglos de fallos
    • El soporte de Control de Versiones ha sido rediseñado. La mayoría de las funcionalidades de integración que fueron deshabilitadas en KDevelop 3.1 están de vuelta en 3.2, junto con un plugin de Subversion muy mejorado.
    • Mejorado "Encontrar en archivos", el soporte de Ctags y el historial de navegación proveen una navegación de código más rápida
    • Se incluyen más plantillas de aplicaciones
    • Nuevas plantillas para aplicaciones y bibliotecas C/C++ basadas en Cmake
    • Nuevas plantillas para aplicaciones KDE basadas en Scons
    • Vea la página del registro de cambios para más detalles
