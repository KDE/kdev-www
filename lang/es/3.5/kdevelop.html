La Versión .0 ha sido liberada con KDE-.8. Las versiones de corrección de errores de KDevelop .x serán liberadas junto a KDE-.x

Errors conocidos y problemas:
  • No hay más modos de interfaz MDI, el único modo disponible es "simpleIDEAl"
  • Qeditor y yzis ya no están soportados. siempre se utilizará KATE.
  • El lenguaje de programación Haskell ya no está soportado.
  • Errores informados en la base de datos bugzilla
Para informar de un nuevo error use el menú de KDevelop Help -> Report Bug... o navegando a través de nuestra página de Errores & Peticiones.
