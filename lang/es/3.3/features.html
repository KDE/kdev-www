Contiene todas las funcionalidades de KDevelop 3.2.x más:
  • Nuevas funcionalidades en la sección de KDevelop en Plan de Funcionalidades de KDE 3.5
  • Ahora puede usar CMAKE para compilar KDevelop
  • Mejora del soporte de lenguaje C++:
    • suporte para QT4, se genera diferente código para proyectos Qt3.x y Qt4.x
    • Se añade una pestaña en el diálogo de opciones de C++ para configurar el uso de la biblioteca Qt. Puede configurar si está usando Qt, cual es la versión principal y donde está instalada la biblioteca Qt.
  • Mejora en el depurador para los lenguajes C, C++ y Ruby:
    • Es más fácil de usar la ventana de puntos de corte del depurador
    • Muestra la ventana de variables automáticamente cuando para la aplicación, y la oculta cuando la aplicación termina.
    • Mejora la edición de los valores de expresión/variable en la ventana de variables.
    • Mejora la vista de las variables
  • Soporte para el lenguaje PHP5
  • Nuevo administrador de proyectos genéricos:
    • Está basado en la arquitectura de plataforma de kdevelop cuando la GUI de administración de proyectos se abstrae para un sistema de construcción específico. Los sistemas de construcción se manejan usando plugins ("importadores de proyecto, constructores y editores")
    • Está disponible el soporte de Workspace
    • Constructores de proyecto: make builder
    • Importadores de proyecto: automake, makefiles propios, qmake
  • vea la página de registros de cambios para más detalle
