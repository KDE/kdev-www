Deze pagina brengt alle informatie over de KDevelop 3.0.x release serie bijeen

Ontwikkeling begon onder de codenaam 'Gideon'. De email A new IDE for a new milleneum :-) (engels) van Bernd Gehrmann beschrijft hoe het allemaal begonnen is.
KDevelop 3.0 is een stabiele release voor KDE-3.x, met een volledig nieuwe code base en vele nieuwe features (o.a. ondersteuning voor nieuwe programmeertalen en nieuwe editors). Versie 3.0.0 is uitgebracht met KDE-3.2.0. KDevelop 3.0.x bugfix versies zullen worden uitgebracht met KDE-3.2.x.
