Sources voor KDE >= :

Laatste ontwikkeling, en daarom onstabiel, snapshot (tar.bz2)

  1. Voor verder te gaan controleer of de nodige software op je computer is geinstalleerd.

  2. Download de source tar.bz2 file.

  3. Pak de source file uit en verander naar de basis directory:
    $ bunzip2 kdevelop.tar.bz2
    $ tar -xf kdevelop.tar
    $ cd kdevelop
