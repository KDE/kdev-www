Voor een meer gedetailleerde lijst met veranderingen bezoek de CVS ChangeLog. Het ondersteunt KDE >= . Was getagd op 05-01-2005 en gereleased op 13-01-2005.
  • NIEUW - Functies voor de KDevelop sectie voor KDE 3.4 Feature Plan
  • NIEUW - KDevelop Platform:
    • De eerste release van het "KDevelop Platform" - een architectuur om IDE-gelijke applicaties te bouwen gebruik makend van gewone plugins en shell
    • KDevelop IDE en KDevelop Assistant zijn nu twee applicaties gebouwd op het KDevelop Platform
    • Herschreven architectuur - herwerkte interfaces en geintroduceerde plugin profielen
  • VERBETERD - KDevelop Shell:
    • Shell is verbeterd voor het gebruik van plugin profielen
    • Notificatie van externe veranderingen in bestanden is verricht
    • Hoofdwindow wordt enkel na initialisatie vertoont
  • VERBETERD - API Documentatie:
    • KDevelop programma API is nu bij http://www.kdevelop.org/HEAD/doc/api/html/index.html
  • VERBETERD - Ruby talen support:
    • Ruby debugger is geimplementeerd
    • Verbeterde ruby parser - meer informatie is verkrijgbaar in class view
    • Qt en KDE applicatie templates toegevoegd
  • GEREPAREERD - kleine bugs in Perl talen support:
  • VERBETERD - Cpp talen support:
    • Debugger breakpoints window vereenvoudigd voor gebruik
    • Aan/uit functies werken nu correct
    • Probleem gemaakt dat breakpoints uitschakeld wanneer deze toch niet uitgeschakeld werden bij het starten van de debugger de eerste keer na het laden van het project
    • Context menu raakt opgekuist
    • Switch header/implementatie kan nu optioneel proberen de momentele functie declaratie/definitie te matchen
    • Code completion database kan verwijderd worden in de configuratie dialoog
  • VERBETERD - PHP talen support: Code completie verbeteringen
  • VERBETERD - Scripting deel:
    • Nieuwe script plugin is toegevoegd
    • KScript gebasseerd scripting is ondersteund om KScriptInterface te gebruiken
    • KJSEmbed, Shell en Kommander kunnen gebruikt worden om een script te starten
  • VERBETERD - Output view:
    • Snelheid optimalisatie (circa 8%)
    • Spring naar de fout locatie werkt nu in alle gevallen
  • VERBETERD - Zoeken in bestanden: Oude zoekresultaten kunnen opgeslagen worden naast de huidige zoektocht
  • VERBETERD - Code snippets plugin: Snippets kunnen verdeeld worden in groepen
  • VERBETERD - Applicatie templates:
    • Installatie, instantiatie and compilatie is hersteld voor volgende applicatie templates: wxhello, kxt, kscreensaver, kdevpart, kdevpart2, kofficepart, kateplugin
    • KDevelop plugin en talen support templates zijn upgedate om de architectuur veranderingen tegemoet te komen
    • Alle voorvallen van verkeerde variabelen substitutie in templates zijn hersteld
    • CMake applicatie templates toegevoegd
  • VERBETERD - Difference viewer:
    • Kate deel is uitgesloten van de lijst van verschillende viewers
    • KDiff3 wordt nu ondersteund
  • VERBETERD - Source formatter: Voorbeeld voor eigen stijl is nu geimplementeerd
  • VERBETERD - Generic projectmanager:
    • Een andere generic project manager was gestart. Het is gebasseerd op de nieuwe architectuur waar de project management GUI is geabstraheerd van het specifieke buildsysteem. Buildsystemen zijn handelende plugins ("project importers, builders and editors")
    • Workspace support is beschikbaar
    • Project builders: make builder
    • Project importers: automake, custom makefiles
  • VERBETERD - QMake project manager:
    • UI is vereenvoudigd
    • Custom qmake variabelen zijn ondersteund
    • Ondersteuning voor libtool archieven, pkgconf bestanden, console apps en windows dlls is toegevoegd
  • VERBETERD - Automake project manager: Ondersteuning voor arbitraire make doelen en commando's in subproject context menu is geimplementeerd
  • GEREPAREERD - Vele fouten in Custom Makefiles project manager
  • VERBETERD - Valgrind ondersteuning: Valgrind 2.1 en hoger worden nu ondersteund(eigen commandolijn opties worden gebruikt)
  • VERBETERD - GUI Designer Integratie: KDevelop is nu compatibel met Kexi Formeditor
  • VERBETERD - KDevelop Designer:
    • Performantie verbeteringen en UI verbeteringen
    • Aanpassen van bron functionaliteit is nu toegankelijk in integratie status
    • Documentatie opties zijn visueel verbeterd
  • VERBETERD - KDevelop Assistant en documentatie plugin:
    • Lettertypen en zoom factoren zijn configureerbaar
    • CHM bestanden worden getoond met het index frame
  • VERBETERD - Algehele verbeteringen:
    • verminderde startop tijd
    • Eigenschap editro bibliotheek verbeteringen
    • XDG menus zijn gecreeerd
    • bron boom is gereconstrueerd en alle ongebruikte onderdelen zijn verwijderd
  • NIEUW - Kyzis editor (CVS versie, www.yzis.org) werkt nu met KDevelop
  • VERWIJDERD - closer, copyto, nedit, buglist, doctreeview, knotes, manager, scripting, sourcenav, cvsclient, cervisia, classbrowser, old generic project manager en ctags plugins zijn verwijderd.
