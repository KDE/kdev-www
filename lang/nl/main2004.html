06 Maart 2004 - Groots herontwerp van de web site. De website speelt een grote rol in KDevelop's ontwikkelings infrastructuue. Dit had vele neven effecten: herwerkte informatie over de oude KDevelop-1.x, W3C HTML eenvormigheid, gerepareerde verloren links, meer gebruik van CSS, gebruikers kunnen een van de vier CSS kiezen, google zoekmachine en niet in het minst een nieuwe kleedje. Hopelijk hou je ervan - Amilcar Lucas.
