Deze pagina bevat alle KDevelop 3.2.x features plus:
  • Nieuwe features bij de KDevelop section op KDE 3.5 Feature Plan
  • Je kan nu CMAKE gebruiken om KDevelop te compileren
  • Verbeterde C++ ondersteuning:
    • ondersteuning voor QT4, Andere code wordt gegenereerd voor Qt3.x en Qt4.x projecten
    • Nieuwe tab toegevoegd in de C++ opties venster om de Qt library te configureren. Je kan, wanneer Qt wordt gebruikt, configureren welke versie wordt gebruikt en waar de Qt library is geinstalleerd.
  • Verbeterde debugger for C, C++ and Ruby languages:
    • Debugger breakpoints venster is gemakkelijker te gebruiken
    • Automatisch wordt de variabelen venster weergegeven wanneer de applicatie stopt, en and hide when application terminates.
    • Implement editing expression/variable values in variables window.
    • Improved the variables view
  • PHP5 language support
  • New generic projectmanager:
    • It's based on the kdevelop platform architecture where project management GUI is abstracted from a specific buildsystem. Buildsystems are handled using plugins ("project importers, builders and editors")
    • Workspace support is available
    • Project builders: make builder
    • Project importers: automake, custom makefiles, qmake
  • see the change log page for more details
