DATALIJN 19 AUGUSTUS 2004

VOOR ONMIDDELLIJKE RELEASE

KDevelop 3.1 release Aankondiging

Het KDevelop Team verstuurt een Grote Feature Release van Award-Winning Open Source IDE

19 Augustus 2004 (Het INTERNET). Het KDevelop Team is trots om KDevelop 3.1, de tweede grote release van de prijzen-winnende Integrated Development Environment (IDE), te versturen.

"KDevelop is een sleutel om nieuwe deuren te open van productiviteit. Het is een IDE dat geschikt is voor elke taak van systeemprogrammering tot GUI RAD ontwikkeling. Integratie van alle standaard UNIX ontwikkelingstools, het zorgen voor support van team ontwikkeling met verschillende versie controle systeemtools en een lange lijst talen ondersteunen, maakt van KDevelop een geschikte keuze voor UNIX ontwikkelaars. Bruikbaar op verschillende platformen zoals Mac OS X en Windows/Cygwin maakt van KDevelop een grootse tool voor multiplatform ontwikkeling. Support voor standaard ontikkelingstools zorgen voor de ontwikkeling van applicaties voor een groot aantal platformen, inclusief embedded systemen." zei Alexander Dymo, project co-maintainer.

"Met zijn nieuwe taal-onafhankelijkhe, plugin-gebasseerde kern en wijde support voor gewone ontwikkelingstools, is KDevelop reeds de de-facto standaard IDE voor Linux/Unix desktops" voegde Harald Fernengel, long-time project member, toe.

"Het KDevelop team heeft schitterend werk verricht met deze release. Meer dan 20 ontwikkelaars droegen in een tijdspan van 6 maanden bij tot het project. Je zal merken dat er moeite is gestoken in het documenteren en in de gebruikersinterface. Deze versie is krachtig, maar toch gemakkelijk te gebruiken" merkte Amilcar do Carmo Lucas, release co-coordinator, op.

Grootse Verbetering

KDevelop ondersteunt nu meer dan 15 programmeertalen en verschillende versie controle systemen, debuggers, documentatie formaten en build tools. Elke Editor die de KTextEditor interface ondersteunt kan gebruikt worden als native KDevelop editor, inclusief kate, QEditor and kvim. Build tools zoals automake, qmake of ant zijn standaard ondersteund, dit betekent dat KDevelop zijn eigen object repository onderhoudt. Veranderingen in de native project files zullen gereflecteerd worden in KDevelop en vice-versa.

De C++ support zorgt voor een class view dat alle symbolen, zowel gewoon als hierarchisch, weergeeft bij namespeces en die het gebruik toelaat van gemakkelijke code navigatie. Een achtergrond parser zorgt voor een on the fly update van de symbolen en geeft ook de syntactische fouten in de broncode weer. De code completion houdt zelfs rekening met Qt signalen en eindes.

Met zijn nieuwe open architectuur kan KDevelop aangepast worden naar ieders wens. De user interface kan makkelijk van classic MDI mode naar de moderne IDEAl mode, die zorgt voor een code-centric aanpak met on-demand tool-views, worden geschakeld.

KDevelop geeft een uitgebreide support voor het navigeren in de broncode. Symbolen en bestandsnamen kunnen benaderd worden door gewoon een stuk van de naam te typen, recent aangepaste bestanden kunnen worden benaderd met een simpele toetsaanslag.

KDevelop biedt documentatie generators, code debugging, geheugen checks, code herschrijving, bladwijzer management en vele tools om ontwikkeling te vereenvoudigen aan. Het is vertaalt in meer dan 16 talen.

Over KDevelop

KDevelop is een onafhankelijk project om sterke en gemakkelijk te gebruiken geintegreerde ontwikkelings omgeving(IDE) te creeeren. Als een stuk van het KDE project deelt KDevelop hetzelfde open ontwikkelings model en hetzelfde modulair framework. Het resultaat is een hoog aanpasbare, taal-onafhankelijke IDE uitgegeven onder de GPL licentie gemaakt door vele ontwikkelaars en professionelen wereldwijd. Meer over KDevelop kan je terugvinden op http://www.kdevelop.org.

Over KDE

KDE is een onafhankelijk project van honderden ontwikkelaars, vertalers, artiesten en andere professionelen wereldwijd die samen werken via het internet om een gesofisticeerde, aanpasbare en stabiele desktop en office omgeving, gebruik makend van een flexibele, component-gebaseerde, netwerk-transparante architectuur, te creeeren en vrij te distribueren en om een uitsteken ontwikkelings platform aan te bieden. KDE zorgt voor een stabiele, volwassen desktop, een volledig, component-gebasseerde office pakkat (Koffice), een groot aantal netwerk en administratie tools en utilities, en een efficente, intuitive ontwikkelings omgeving gebruik makend van de excellente IDE KDevelop. KDE bewijst dat het Open Source "Bazaar-stijl" software ontwikkelings model best-genoteerde technologieen kan laten concureren met de meest complexe commerciele software.
