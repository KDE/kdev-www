Deze pagina bevat alle informatie over de KDevelop 1.x release series

De ontwikkeling van KDE 1.x is beeindigd op 11 December 2000 met de KDevelop 1.3 release, en er zijn geen verdere KDevelop 1.x releases gepland voor KDE 1.x.
KDevelop 1.4 is bedoeld voor KDE 2.x and Qt 2.x. maar gebruik a.u.b. KDevelop 2.1.5 in de plaats als je een KDevelop nodig hebt voor KDE 2.x.
Er blijven momenteel nog maar vier wensen over in KDevelop 3.0 (voor KDE 3.0, Qt 3.0) van de vele bugs en wensen van KDevelop 1.3.

Compileren van versie 1.3 vanuit de CVS repository

Om een zelf gecompileerde KDevelop 1.3 uit de CVS sources te bekomen, doe een checkout van de CVS module "kdevelop" met als CVSROOT :pserver:anonymous@anoncvs.kde.org:/home/kde gebruik makende van de CVS branch KDEVELOP_1_0

vanop de command line als je KDE 1.x draait:
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Om KDevelop te kunnen compileren en installeren op jou systeem,
moet je de volgende commando's vanuit de basis directory van de KDevelop distributie typen:

een release-gecompileerde versie: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (as root)

een debug-gecompileerde versie: 
       $ export KDEDIR=/where/your/kde1/is
       $ export QTDIR=/where/qt-1/is
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (as root)
