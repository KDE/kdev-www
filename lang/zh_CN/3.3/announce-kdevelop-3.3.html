最新版

KDevelop 发布

KDevelop小组发布包含主要功能的屡获殊荣的开放源代码的集成开发环境（IDE）

2005年11月29日（互联网), KDevelop开发小组自豪的发布了 KDevelop 的第三版，KDevelope是屡获殊荣的开放源代码的集成开发环境（IDE）.

发行副协调人Amilcar do Carmo Lucas说到“：超过30多个开发人员，长达七个多月的开发工作后,我们的用户数正在稳步增长,每小时的平均下载量是以往的二十倍,这也为我们提供了许许多多关于如何改进程序的信息.”

主要改进

这次主要的改进是提高了调试器和支持新的Qt4。

目前,KDevelop支持多达15种程序设计语言,众多版本控制系统,调试器,文档格式及编译工具.任何支持KTextEditor接口的编辑器都可以当做KDevelop的代码编辑器,其中包括Kate,QEditor 和 Yzis. 编译工具像automake,qmake 和ant得到良好的内置支支持，这意味着KDevelop不需要维护它自己的对象库,项目文件的变化将在KDevelop中反映，反之亦然.

KDevelop的类视图,可以按名字空间显示平面或者层叠的符号表,十分便于代码浏览.后台的解析器在快速更新符号表,并显示源代码中的语法错误.代码补齐功能甚至可以处理Qt的信号(signal)和槽(slot).

由于它的开发构架,KDevelop可以在各个方面定制及扩展.核心的集成开发环境（IDE）动能已经做成一套叫"KDevelop platform"的库,这套库可以用来做新的集成开发环境（IDES）的基础部件.用户界面可以从经典的MDI模式转换到现代的IDEAl模式:以代码为中心的特性及按需的工具视图的方法.

KDevelop的特色是广泛支持源代码导航:符号和文件名可以通过输入名字的部分来访问,最近修改的文件也可以通过简单的击键来访问.

KDevelop提供了许多文档生成器,代码调试器,内存检测,代码重构,书签管理以及成打的其他工具,使得开发的过程更加方便快捷,它被翻译成17种语言.

安装KDevelop 二进制代码包

所有主要Linux发行版的KDevelop二进制代码包可以在以下网站找到 http://www.kdevelop.org/index.html?filename=/download.html.

Mac OS X原生支持的alpha版可以在以下网站找到 http://kde.opendarwin.org.

kde-cygwin项目在以下网站提供了KDevelop http://kde-cygwin.sourceforge.net/kde3/installation.php.

请注意,KDevelop没有发布任何二进制版本,所有的发行包都是由第三方提供的.

编译KDevelop

完整的 KDevelop 源代码可以从以下网站自由的下载 http://www.kdevelop.org. 详细的编译信息可以在此找到 http://www.kdevelop.org/index.html?filename=/branches_compiling.html.

关于 KDevelop

KDevelop是一个独立的项目，用于产生一个功能强大又易于使用的集成开发环境(IDE)。 作为KDE项目的一部分，KDevelop采用了同样的开放的开发模式和模块化框架。这样就在世界范围内成批的开发者和专业人士在GNU通用公共许可证的驱动下，形成了一个非常用户化的，语言独立的集成开发环境。如果你想更多了解关于KDevelop，可以访问: http://www.kdevelop.org.

About KDE

KDE是一个由全世界上百个程序员，翻译者，艺术家和其他专业人士通过互联网协作的独立项目，采用了灵活的，基于组件的，网络透明的架构，产生一个可以自由发布，稳定的，可订制桌面和办公环境，并提供了一个出色的开发平台。KDE提供了一个稳定，成熟的桌面，完整的基于组件的办公套件(KOffice)，一系列网络和管理工具，以及高效的，内置的开发环境——KDevelop。KDE证明了开源社区“大教堂”软件开发模式一样可以产生一流的技术，可以和最复杂的商业软件相比，甚至更好。
