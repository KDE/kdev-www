06 Mars 2004 - Importante refonte du site web. Le site web joue une part importante dans l'infrastructure du développement de KDevelop. L'objectif principal était de permettre d'avoir des pages web indépendantes par version. Cela rend possible la présentation de l'information de façon claire sur les versions de KDevelop stable (3.0.x) et en développement (HEAD).
Ceci a plusieurs conséquences : réssusciter l'information sur l'ancien KDevelop-1.x, valider le code HTML selon le W3C, corriger les liens brisés, utiliser plus de CSS, les utilisateurs peuvent choisir l'une des quatre CSS, ajout d'un moteur de recherche google et d'un moteur de recherche dans le site web et enfin et non des moindres, un nouveau look. J'espère que vous aimez - Amilcar Lucas.
