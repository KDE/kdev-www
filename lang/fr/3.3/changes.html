Il supporte KDE >= . La branche a été faite le 26-07- 2005 et il n'y a pas eu de sortie. Il contient les nouvelles caractéristiques de KDevelop plus les changements suivants :
  • NOUVEAU - Features on the KDevelop section on KDE 3.5 Feature Plan
  • NOUVEAU - Vous pouvez maintenant utiliser CMAKE pour compiler KDevelop
  • AMÉLIORÉ - support du language Cpp :
    • support pour QT4
    • Ajout d'un nouvel onglet dans le dialogue des options C++ pour configurer l'utilisation de la bibliothèque Qt. Vous pouvez choisir quel Qt est utilisé, quelle version et déterminer le chemin vers le répertoire d'installation.
    • Assistant de classe : - Les boîtes à cocher Qt sont invalidées pour des projets non Qt. Du code différent est généré pour des projets Qt3.x et Qt4.x
  • AMÉLIORÉ - Débogueur :
    • Fenêtre des points d'arrêt du débogueur plus facile à utiliser
    • Correction du problème avec les points d'arrêt non utilisés
    • Affiche automatiquement la fenêtre des variables quand l'application stoppe et la cacher lorsque l'application se termine.
    • Implémente l'édition des valeurs des expressions/variables dans la fenêtre des variables.
    • Amélioration de la vue des variables
  • NOUVEAU - support de language PHP5, config localise l'exécutable php
  • AMÉLIORÉ - Visualiseur de sortie :
    • Optimisations de vitesse
  • AMÉLIORÉ - Gestionnaire de projet générique :
    • Il est basé sur l'architecture de la plate-forme kdevelop dans laquelle l'interface graphique du gestionnaire de projet est adaptée à un système de construction spécifique. Les systèmes de construction sont gérés par des modules externes ("importateurs de projets, constructeurs et éditeurs")
    • Support d'espace de travail maintenant disponible
    • Constructeurs de projets : constructeur make
    • Importateurs de projets : automake, custom makefiles, qmake
  • AMÉLIORÉ - Manuel de l'utilisateur
  • CORRIGÉ - environ 48 bogues des séries KDevelop 3.2.x
