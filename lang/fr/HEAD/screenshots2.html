Fenêtre des variables Menu conceptuel principal
  • Les variables locales seulement sont affichées, il n'y a plus de confusion sur les variables générales
  • Il y a un nouveau bouton "Evaluate" qui permet d'évaluer l'expression sans la mettre à jour dans l'étape suivante.
  • Si vous regardez la variable "p1" ou "p3" : la valeur n'inclut plus le type comme préfixe. Avant il y avait "(int*) 0x8049abc" pour la valeur.
  • Si vous regardez les variables "p4" et "s" : il est maintenant possible d'afficher les membres statiques d'une classe sans casser l'affichage deQString
  • "Run to cursor" est ajouté
  • "Evaluate" est ajouté
  • Quand le débogueur est actif, les commandes de débogage sont en haut
