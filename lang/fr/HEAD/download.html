Sources pour KDE >=  :

Dernier développement et par conséquent version instable (tar.bz2)

  1. Avant de continuer, veuillez vérifier que votre ordinateur a bien tous les logiciels requis installés.

  2. Téléchargez le fichier source tar.bz2.

  3. Décompressez le fichier source et allez dans le répertoire de base :
    $ bunzip2 kdevelop.tar.bz2
    $ tar -xf kdevelop.tar
    $ cd kdevelop
