Bogues connues et problèmes:
  • Finis les modes interface MDI, le seul mode disponible est "simpleIDEAl"
  • Qeditor et yzis ne sont plus disponibles. KATE est toujours utilisé.
  • Le language de programmation Haskel n'est plus supporté.
  • Bogues rapportées dans la base de données
Pour rapporter une nouvelle bogue, veuillez utiliser le menu de KDevelop Aide -> Rapport de bogue... ou naviguez dans notre page de bogues & souhaits.
