Cette page regroupe toutes les informations concernant les versions de KDevelop 3.0.x

Le développement a débuté sous le nom de code Gideon. Le courriel Une nouvelle Interface de Développement pour un nouveau millénaire :-) de Bernd Gehrmann décrit comment il a débuté.
KDevelop 3.0 est stable pour KDE-3.x, basé sur un code entièrement nouveau, beaucoup de nouvelles caractéristiques (par exemple le suppoort de nouveaux languages de programmation, de nouveaux éditeurs). La version 3.0.0 a été fournie avec KDE-3.2.0. Les versions de corrections de bogues de KDevelop 3.0.x sortiront avec KDE-3.2.x
