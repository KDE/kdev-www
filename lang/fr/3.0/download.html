Exécutables :

Certains vendeurs de distributions Linux/UNIX fournissent des paquetages exécutables de KDevelop pour certaines versions de leur distribution et dans d'autres cas, des volontaires le font aussi. ertains de ces paquetages binaires sont disponibles en téléchargement libre depuis les miroirs KDE http ou FTP.



Sources pour KDE >= 3.0 :


Avant de continuer, veuillez vérifier si votre ordinateur possède les logiciels requis.

Décompressez le fichier source et allez dans le répertoire de base : 
       $ bunzip2 kdevelop-3.0.x.tar.bz2
       $ tar -xf kdevelop-3.0.x.tar
       $ cd kdevelop-3.0.x
