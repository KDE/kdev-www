• Architecture basée sur des modules externes
    • Des fonctionalités peuvent être modifiées, ajoutées ou supprimées sans changer le code source fondamental.
    • La licence des bibliothèques permet d'écrire des modules commerciaux et libres pour l'IDE.
    • Support pour les profils qui déterminent un ensemble de modules externes qui seront chargés. Ceci est spécialement utile dans le cas d'environnements restreints ou pour des configurations personnalisées.
  • Gestionnaire de projet
    • Support natif pour automake et qmake. Des modifications manuelles sont permises et reconnues par l'IDE, des projets existants ainsi que leurs sous-projets peuvent aussi facilement être importés.
    • Vous pouvez personnaliser vos makefiles pour compiler le projet.
    • Support pour ant - outil de compilation pour les projets java.
    • Support pour des types de projets tels pascal, haskell, ada et des projets de scripts.
    • Cadre pour ajouter le suppport pour tout système de compilation en créant des modules externes de systèmes de compilationpour le gestionnaire de projet standard.
    • Nouvel assistant de fichier pour gérer les squelettes de fichiers et créer de nouveaux fichiers dans le projet depuis des squelettes.
    • Les modules externes du compilateur définissent les options du compilateurs dans des dialogues pratiques.
    • Profils de configuration avec la capacité de spécifier des compilateurs différents, des drapeaux de compilation, les répertoires de compilation, l'architecture cible, etc.
    • La documentation des projets est facilitée avec par exemple le support de doxygen pour créer une documentation API automatique (dans les formats html, tex et man) pour un projet avec une référence croisée des bibliothèques.
    • Support pour l'internationalisation, permettant aux traducteurs d'ajouter leur langue cible facilement dans un projet, incluant le support pour KBabel.
    • Support pour les projets Qt/Embedded (pour le Zaurus et les plates-formes iPAQ.
  • Support de languages
    • 15 languages supportés incluant Ada, C, C++, Objective-C (via support C), SQL, Fortran, Haskell, Java, PHP, Pascal, Perl, Python, Ruby, Bash, XUL (non officiellement).
    • Rapporteur de problèmes qui rapporte divers problèmes avec le code source pendant que vous tapez. Ces problèmes incluent les erreurs de syntaxe, les TODO et FIXME.
    • Navigateur de classes qui affiche la hiérarchie des classes et autres symboles du projet.
    • Analyseurs de language pour le navigateur de classes et rapporteur de problèmes.
    • Assistant de classe, de méthode et d'attribut pour les langugaes orientés object (OO).
    • Complétion de code automatique et suggestion pour les variables de classe, les méthodes, les arguments des fonctions et plus.
    • Formatage des sources, surbrillance syntaxique et déploiement du code.
    • Caractéristiques spécifiques à C++ telles que
      • "créer un membre" - créer la définition des méthodes dans le fichier source en se basant sur la déclaration de la méthode dans le fichier en-tête;
      • "extraire l'interface"- extraire l'interface (déclarations des méthodes) et créer une classe de base avec cette interface;
      • "surclasser" - création automatisée du surclassement et de l'implentation des slots des formes de Qt Designer.
  • Débogueur intégré
    • Interfaces pour les débogueurs GDB et Java .
    • Console GDB intégrée.
    • Breakpoints et watchpoints pour des lignes, addresses et méthodes.
    • Points de rupture conditionnels.
    • Désassembleur et vue du stack.
    • Arborescence pour suivre une variable.
    • Visualiseur de la mémoire, des registres et des bibliothèques.
    • Support du débogueur shell (libtool).
    • Déboguage à distance.
  • Édition du code
    • Tout éditeur qui utilise l'interface KTextEditor de KDE peut être utilisé avec KDevelop.
    • Les éditeurs actuellement disponibles sont Kate, QEditor (basé sur l'éditeur de Qt Designer) et KVim.
    • Les signets du projets sont sauvegardés dans le fichier du projet.
    • Visualiseur de différence entre le texte dans l'éditeur et celui sur le disque dur.
    • Visualiseur de différence pour les fichiers patch.
    • Intégration de grep - recherche d'expression régulières dans le projet.
    • Outil de recherche et de remplacement dans tout le projet.
  • Système de contrôle de version
    • API commune pour inclure le support VCS dans KDevelop.
    • Les systèmes VCS supportés : CVS, Perforce, Subversion et Clearcase.
  • Assistant d'application
    • L'assistant d'application génère des applications examples toutes prêtes.
    • Des squelettes existent pour chaque language supporté et chaque compilateur.
    • Possibilité d'utiliser des squelettes d;application personnelles non inclus dans la distribution.
    • Des squelettes de projets reliés à KDE (applications KPart, simple applications KDE, modules de KControl, applets Kicker (panneau), parts KOffice, KIOSlaves, modules externes Konqueror, Noatun et Kate et styles).
  • Visualiseur de documentation
    • Interface unifiée pour naviguer dans la documentation.
    • Possibilité de TOC, index et recherche dans tout le texte (avec htdig).
    • Autodétection et inclusion de la documentation Qt, KDE Libset DevHelp.
    • Support pour des ensembles de documentation personnelle (format KDevelopTOC).
    • Possibilité d'ouvrir de la documentation locale, à distance ou encore compressée (en utilisant tout protocole supporté par Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Possibilité d'ouvrir divers formats de documentation (html, text, ps, pdf, chm, etc.).
  • Autres caractéristiques
    • L'interface utilisateur est totalement configurable, elle permet la navigation dans les classes à la volée dans les modes MDI comme toplevel, childframe et le nouveau mode IDEAl qui permet des vues "glissées".
    • Console intégrée.
    • Sélectionneur de fichiers, arborescence de fichiers et groupements de fichiers suppriment le besoin d'un gestionnaire de fichiers externe.
    • Inclusion de tout autre programme dont vous avez besoin pour le développement en l'ajoutant au menu Outils selon vos besoins personnels.
    • Test d'expressions régulières.
    • Intégration de Valgrind pour dépister les fuites de mémoire dans les programmes.
    • Distribution du projet et outil d'empaquetage pour créer automatiquement les paquetages sources et binaires.
    • Intégration de CTags.
    • Une part explorer pour questionner tous les services disponibles de KDE.
