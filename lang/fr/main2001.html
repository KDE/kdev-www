22 Octobre 2001 - Jeudi dernier, KDevelop a gagné le Linux New Media Award 2001 pour le "Meilleur outil de développement Linux". :-) Ralf Nolden a accepté le prix remis par le Linux New Media AG à la Systems fair à Munich, Allemagne. Nous avons donné l'argent (1500 Euros) au compte de l'organisation KDE.

20 Févrierr 2001 - Anne-Marie Mahfouf a écrit un tutoriel sympa sur l'utilisation de KDevelop1.4 et Qt designer. Il est publié sur le site web women.kde.org

02 Février 2001 - Avec la sortie de KDE2.1beta2, il y a aussi une nouvelle mise à jour de KDevelop. KDevelop1.4beta2 est surtout un port de KDevelop1.3 port pour KDE2.1 mais contient aussi de petites améliorations comme des widgets ancrés. Vous pouvez obtenir le code source et les versions exécutables dans notre section téléchargement ou sur tout miroir ftp KDE.

07 Janvier 2001 - Stefan Bartel a créé un style KDE2 inspiré par le site web KDevelop. Vous pouvez le télécharger depuis kde.themes.org. Vraiment cool!

04 Janvier 2001 - KDevelop 1.4 pour KDE2.1. Deux semaines après la dernière annonce, c'est ce que nous avons produit.
