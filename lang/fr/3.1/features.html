Il contient toutes les caractéristiques de KDevelop 3.0.x plus:
  • Bouts de code prédéfinis (Code Snippets)
    • SnippetPart ajoute une vue-outil qui s'ancre par défaut sur la droite
    • Ajout, édition et suppression de bouts de codes par un menu contextuel
    • Double-cliquer sur un bout de code à partir de la liste l'insère dans la vue active et à la position du curseur
    • Des info-bulles montrent le contenu d'un bout de code
    • Les bouts de code sont enregistrés dans le répertoire home de l'utilisateur donc, chaque utilisateur peut avoir ses propres bouts de code
    • Les bouts de code peuvent contenir des variables du genre $VARNAME$. Quand on utilise le bout de code, on demande à l'utilisateur d'entrer la valeur de remplacement pour les variables
  • Intégration Doxygen (générateur de documentation API)
    • Supporte tous les langages de programmation que doxygen supporte (sauf IDL et D)
    • Prévisualisation de la documentation que doxygen génère pour le fichier que vous êtes en train d'éditer
    • Configuration de doxygen
    • Exécution de doxygen
    • Affiche les docs. générées par doxygen (formats html man ou pdf)
    • On peut utiliser le XML généré par doxygen comme info-bulles pour l'achèvement automatique du code "lors de la frappe"
  • Visualisateur de documentation basé sur des plugins
    • Une interface unique pour afficher la documentation.
    • Table des matières, index et possibilité de recherche complète de texte (avec htdig).
    • Autodétection et inclusion de la documentation Qt, KDE Libs et DevHelp .
    • Support pour des collections de documentation personnalisée (format KDevelopTOC).
    • Possibilité d'ouvrir de la documentation locale, distante et même compressée (en utilisant n'importe que protocole supporté par Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Supporte les formats de fichiers TXT, HTML, DevHelp, CHM, PDF, Djvu et PDB.
    • Peut utiliser n'importe quel format de documentation que supporte KDE
  • Assistant d'application flexible (modèles de projets)
    • L'assistant d'application génère des applications complètes, et prêtes à l'emploi.
    • Des modèles pour chaque langage supporté et système de construction existent.
    • Possibilité d'utiliser des modèles d'application personnalisés non inclus dans la distribution.
    • Contient tous les modèles KDevelop 3.0 plus les modèles d'application GTK, GNOME et Bonobo
    • Supporte des choses que seul Ian connaît :)
  • Orienté RAD (Développement Rapide d'Applications)
    • La nouvelle application kdevdesigner permet le développement d'interfaces basées sur QT par l'intégration de QTdesigner dans le cadre de KDE.
    • La nouvelle application kdevassistant permet des aides de documentation style QT.
    • Ces deux applications peuvent être utilisées d'une manière autonome ou à partir de KDevelop lui-même, ce qui permet un programme de développement pour tous vos besoins de développements.
