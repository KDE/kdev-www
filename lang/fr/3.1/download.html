Exécutables :

Certains vendeurs de distributions Linux/UNIX fournissent des paquetages exécutables de KDevelop pour certaines versions de leur distribution et dans d'autres cas, des volontaires le font aussi. Certains de ces paquetages binaires sont disponibles en téléchargement libre depuis les mirroirs KDE http ou FTP.



Sources pour KDE >= 3.2 :


  1. Avant de continuer, veuillez vérifier si votre ordinateur possède les logiciels requis insallés.

  2. Décompressez le fichier source et allez dans le répertoire de base :
    $ bunzip2 kdevelop-3.1.x.tar.bz2
    $ tar -xf kdevelop-3.1.x.tar
    $ cd kdevelop-3.1.x
