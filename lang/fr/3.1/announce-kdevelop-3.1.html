Annonce de la sortie de KDevelop 3.1

L'équipe de KDevelop sort une version majeure de leur EDI récompensé

19 Août 2004 (INTERNET). L'équipe de KDevelop est fière de livrer KDevelop 3.1, la deuxième version majeure de son Environnement de Développement Intégré (EDI) récompensé.

"KDevelop est une clef pour atteindre de tout nouveaux niveaux de productivité. C'est un EDI qui convient pour toutes les tâches de la programmation système au développement d'interfaces RAD. En intégrant tous les outils standards de développements UNIX, en proposant du support pour le développement en équipe avec divers outils de contrôle de version et en ayant une longue liste de langages supportés, KDevelop est un excellent choix pour les développeurs UNIX. Disponible sur différentes plateformes comme Mac OS X et Windows/Cygwin, KDevelop est un bon outil pour le développement multiplateforme. Le support pour des outils de développement communs vous permet de développer des applications pour un grand nombre de plateformes y compris les plateformes embarquées" déclara Alexander Dymo, co-mainteneur du projet.

"Avec son nouveau noyau indépendant du langage, basé sur des plugins et son large support pour les outils de développement communs, KDevelop est déjà l'EDI standard de-facto sur les stations Linux/Unix" ajouta Harald Fernengel, membre du projet de longue date.

"L'équipe de KDevelop a fait un excellent travail sur cette version. Plus de 20 développeurs ont contribué au projet sur une durée de six mois. Vous remarquerez qu'une attention particulière a été prise pour la documentation et l'interface utilisateur. Cette version est puissante, et maintenant facile à utiliser" nota Amilcar do Carmo Lucas, co-coordinateur de version.

Améliorations majeures

KDevelop supporte maintenant plus de 15 langages de programmation et de nombreux systèmes de contrôle de version, débogueurs, formats de documentation et outils de construction. N'importe quel éditeur supportant les interfaces KTextEditor peut être utilisé comme éditeur KDevelop natif, y compris kate, QEditor et kvim. Les outils de constructions comme automake, qmake ou ant sont supportés nativement, ce qui veut dire que KDevelop ne garde pas sa propre collection d'objets. Les changements dans les fichiers du projet natif se réfléchiront dans KDevelop et vice-versa.

Le support C++ offre une vue de classe montrant tous les symboles sous forme de liste ou sous forme hiérarchique par ordre alphabétique et permet une navigation aisée dans le code source. Un parseur met à jour les symboles à la volée et montre aussi les erreurs syntaxiques dans le code source. L'achèvement automatique du code gère aussi les slots et signaux Qt.

Avec sa nouvelle architecture ouverte, KDevelop peut être personnalisé sur tous les points. L'interface utilisateur peut être changée en temps réel du mode classique MDI au mode moderne IDEAl, avec une approche centrée sur le code avec des vues-outils sur demande.

KDevelop offre un support étendu pour naviguer dans le code source. Les symboles et noms de fichier sont accessibles en ne tapant qu'une partie du nom, les fichiers récemment modifiés peuvent être atteints avec une simple combinaison de touches.

KDevelop offre des générateurs de documentation, des débogueurs de code, des vérificateurs de mémoire, un arrangeur de code, une gestion de signets, et des tonnes d'outils pour rendre le développement facile et convivial. Il est traduit dans plus de 16 langues.

Installer les paquetages binaires de KDevelop 3.1

Les paquetages binaires pour toutes les distributions majeures de Linux se trouvent sur http://www.kdevelop.org/index.html?filename=3.1/download.html.

Une version alpha d'un binaire natif pour Mac OS X se trouve sur http://kde.opendarwin.org.

Le projet kde-cygwin propose KDevelop sur http://kde-cygwin.sourceforge.net/kde3/installation.php.

Veuillez noter que l'équipe de KDevelop ne fournit pas de versions binaires, tous les paquetages sont mis à disposition par des tiers.

Compiler KDevelop 3.1

Le code source complet de KDevelop 3.1 peut être librement téléchargé sur http://www.kdevelop.org. Les informations détaillées sur la construction se trouvent sur http://www.kdevelop.org/index.html?filename=3.1/branches_compiling.html.

A propos de KDevelop

KDevelop est un projet indépendant pour créer un Environnement de développement intégré (EDI) puissant et facile à utiliser . Faisant partie du projet KDE, KDevelop partage le même modèle de développement ouvert et le cadre modulaire. Le résultat est un EDI extrêmement personnalisable et indépendant du langage sous licence GPL suivie par des dizaines de développeurs et de professionnels dans le monde. Pour en savoir plus sur KDevelop, visitez http://www.kdevelop.org.

A propos de KDE

KDE est un projet indépendant de centaines de développeurs, traducteurs, artistes et d'autres professionnels collaborant à travers le monde grâce à Internet pour créer et distribuer gratuitement une station et un environnement de bureau sophistiqués, personnalisables et stables en employant une architecture flexible, basée sur des composants, transparente pour le réseau et offrant une plateforme de développement exceptionnelle. KDE propose un bureau stable, mûr, une suite bureautique complète, et basée sur des composants (KOffice), un grand choix d'outils et d'utilitaires d'administration et de réseau, et un environnement de développement efficace et intuitif: l'excellent EDI KDevelop. KDE est la preuve vivante que le modèle de développement de logiciel open source "style Bazaar" peut égaler et même dépasser les technologies de premier ordre avec les logiciels commerciaux des plus complexes.
