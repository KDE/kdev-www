Cette page rassemble toutes les informations sur les séries KDevelop 1.x

Le développement de KDevelop 1.x a été arrêté le 11 Décembre 2000 avec la sortie KDevelop 1.3, et aucune autre sortie de KDevelop 1.x n'est prévue pour KDE 1.x.
KDevelop 1.4 est pour KDE 2.x et Qt 2.x. mais veuillez s'il vous plait utiliser KDevelop 2.1.5 si vous avez besoin d'un KDevelop pour KDE 2.x.
Il ne reste actuellement que 4 souhaits pour KDevelop 3.0 (pour KDE 3.0, Qt 3.0) sur toutes les bogues et les souhaits de KDevelop 1.3.

Cette page vous donne un aperçu des caractéristiques de KDevelop 1.x.

Compiler la version 1.3 du CVS

Pour obtenir un KDevelop 1.3 compilé et débogué depuis les sources cvs, faire un checkout du module CVS "kdevelop" à :pserver:anonymous@anoncvs.kde.org:/home/kde en utilisant la branche CVSdes corrections de bogues KDEVELOP_1_0

ligne de commande si vous avez KDE 1.x :
       $ mkdir kdesrc
       $ cd kdesrc
       $ cvs -z3 -d :pserver:anonymous@anoncvs.kde.org:/home/kde co -r KDEVELOP_1_0 kdevelop
       $ cd kdevelop

Pour compiler et installer KDevelop (version CVS) sur votre système,
veuillez entrer les commandes ci-dessous dans le répertoire de base de KDevelop :

une version compilée : 
       $ export KDEDIR=/où/est/votre/kde1
       $ export QTDIR=/où/est/votre/qt-1
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --prefix=$KDEDIR
       $ make
       $ make install (en tant que root)

une version déboguée-compilée : 
       $ export KDEDIR=/où/est/votre/kde1
       $ export QTDIR=/où/est/votre/qt-1
       $ export LD_LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LD_LIBRARY_PATH
       $ export LIBRARY_PATH=$QTDIR/lib:$KDEDIR/lib:$LIBRARY_PATH
       $ export PATH=$QTDIR/bin:$KDEDIR/bin:$PATH

       $ export CXXFLAGS="-g3 -O0"

       $ make -f admin/Makefile.common cvs-clean
       $ make -f Makefile.cvs
       $ ./configure --enable-debug --prefix=$KDEDIR
       $ make
       $ make install (en tant root)

Bogues connues

Comme le développement a été arrêté pour cette série, il reste beaucoup de bogues connues non corrigées. La bonne nouvelle est que toutes les bogues sont corrigées dans KDevelop 3.0. Veuillez donc utiliser la version 3.0 à la place.
