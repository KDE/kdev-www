Nouvelle version de l'IDE C/C++ KDevelop pour le bureau KDE

La nouvelle version du puissant environnement de développement intégré C/C++ pour Linux et autres UNIXes est mise à disposition

11 Décember 2000 (l'INTERNET). L'équipe KDevelop a annoncé aujourd'hui la diffusion de KDevelop 1.3, un environnement de développement intégré puissant, facile à utiliser, pour le langage de programmation C et C++, basé sur KDE. KDevelop 1.3 est une mise à jour majeure qui suit les versions 1.1 et 1.2 couronnées de succès et qui permet aux développeurs un développement rapide d'applicationspour le nouveau bureau KDE 2.0. KDevelop est le fruit du travail d'une équipe internationale qui a pour vocation de fournir un IDE pour les systèmes d'exploitation Linux et UNIX, leur produit disponible gratuitement est connu pour être stable et améliorer la productivité pour tout développeur C/C++.

Les objectifs principaux de la version 1.3 sont de simplifier le processus de développement de nouvelles applications pour le bureau UNIX KDE 2.0, de permettre aux entreprises de déployer facilement l'API de KDE 2.0 et de faciliter leur décision de créer des programmes qui sont compatibles avec le bureau UNIX et la dernière version de KDE.

Plus précisément, les nouveaux outils de développement fournis par Trolltech, le Qt Designer et le prochain Qt Linguist, ainsi que le programme d'internationalisation pour KDE KBabel sont directement supportés et intégrés dans le processus de développement. Une autre caractéristique majeure est le générateur de fichiers pour le projet inclu dans KDevelop, qui permet de travailler facilement sur des projets automake déjà existants avec KDevelop en quelques secondes et qui va apporter tout son sens aux licences GPL et autre Logiciels Libres dans leur but de réutilisation du code chaque fois que possible. Les développeurs peuvent maintenant regarder le code source complet de KDE etd e tout autre projet pour examiner le code de la fonctionalité qu'ils veulent apporter dans leur application. La disponibilité de l'API Qt et KDE à l'intérieur de l'environnement de développement simplifie et réduit les efforts que les dévelopeurs doivent mettre pour trouver l'information dont ils ont besoin pour programmer sous des aspects temps/demande.

Ben que KDevelop 1.3 soit encore basé sur le code de KDE 1.1.2, l'équipe fournit une version binaire qui a été testée pour tourner sur la plupart des distributions principales de Linux sans que les bibliothèques KDE 1.1.2 soient installées. Ainsi, les développeurs dont les systèmes ne permettent pas de compiler des applications KDE 1.1.2 peuvent quand même utiliser simplement l'IDE C/C++ IDE pour un développement rapide d'applications.

L'IDE KDevelop 1.3 est disponible gratuitement et librement sous la Licence Publique Générale GNU (GPL) tout comme KDE est disponible sous les licences Open Source. De la même façon, Qt 2.2.2 de Trolltech, la boîte à outils GUI sur laquelle est basé KDE, est aussi disponible librement sous deux licences Open Source : la Licence Publique Q et la Licence Publique Générale GNU.

Plus d'information sur KDevelop 1.3 est disponible sur le site web de KDevelop et sur le site web de KDE, ainsi que de l'information pour les développeurs et un Guide pour porter KDE 1 - KDE 2

Télécharger et compiler KDevelop

Les paquetages source pour KDevelop 1.3 sont disponibles pour un téléchargement libre à ftp://ftp.kdevelop.org/pub/ kdevelop/ ou à l'un des nombreux miroirs de serveurs ftp KDevelop. KDevelop 1.3 a besoin de qt-1.44, qui est disponible au serveur ftp de TrollTech à ftp://ftp.troll.no et des bibliothèques KDE 1.1.2, qui sont disponibles aux serveurs ftp KDE à ftp://ftp.kde.org pour la compilation, pour développer des applications, vous pouvez vous sentir libre de choisir entre les versions de KDE 1.1.2 ou de KDE 2.0 et entre les bibliothèques Qt 1.44 ou Qt >2.2.1 car les deux sont directement supportées.

Pour plus d'instructions sur la compilation et l'installation de KDevelop, veuillez consulter les instructions d'installation fournies avec le paquetage source de KDevelop.

Installer les paquetages binaires

Certains distributeurs ont déjà choisi de ne fournir que des paquetages de KDE 2.0 pour certaines versions de leur distribution. Comme cela empêche de compiler KDevelop à cause de l'absence d'une installation de KDE 1.1.2, l'équipe KDevelop fournit une version binaire qui est testée pour fonctionner sous la plupart des versions de Linux. La version binaire de KDevelop est donc disponible pour un téléchargement gratuit à ftp://ftp.kdevelop.org/pub/kdevelop/ ou sous le répertoire équivalent à l'un des nombreux sites ftp miroirs de KDevelop.

A propos du projet KDevelop

Le projet KDevelop, qui a débuté l'été 1998, a pour objectif de fournir aux développeurs un bon environnement stable et utile qui peut concurrencer les IDEs modernes que l'on trouve sur d'autres systèmes d'exploitation. Depuis les débuts du développement, l'équipe a grossi de trois membres au départ dans l'équipe de base jusqu'à plus de dix codeurs et une équipe de translation de plus de 50 personnes, toutes donnant de leur temps libre pour travailler sur ce projet. Dans le même temps, beaucoup d'utilisateurs heureux de l'IDE Kdevelop se sont joints au projet et ont partagé leur expérience pour améliorer KDevelop en stabilité et en fonctionalité.

Pour plus d'information sur KDevelop, veuillez visiter le site web de KDevelop.
