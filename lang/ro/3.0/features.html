• Arhitectura bazata pe plugin
    • Functionalitate poate fi adaugata, inlocuita si scoasa fara a altera codul sursa.
    • Licenta librariilor (LGPL) permite crearea de plugin-uri IDE comerciale si gratis.
    • Suport pentru profile care determina un set de plugin-uri sa fie incarcate. Aceasta este folositor in special pentru medii restrictionate sau setari customizate.
  • Project Management
    • Nativ non-intrus suport automake si qmake. Modificari facute manual permise si recunoscute de IDE, de asemenea proiecte si subproiecte existente care pot fi usor importate.
    • Abilitate de a folosi makefile-uri pentru a construi proiectul.
    • Suport pentru ant - utilitar de constructie pentru proiecte java.
    • Suport pentru tipuri de proiect ca pascal, haskell, ada si proiecte de scripting.
    • Suport pentru orice sistem de construire creand pluginuri de sistem pentru un project manager general.
    • Wizard de fisier nou pentru a administra modele de fisiere si crea fisiere noi in proiect din modele.
    • Plugin-uri de compilator pentru a defini optiunile compilatorului in dialoguri convenabile.
    • Profile de configurare cu abilitatea de a specifica diferite compilatoare, parametri de compilare, directoare de build, arhitectura tintita etc.
    • Facilitati de documentatie a proiectului cum ar fi suport doxygen pentru crearea de documentatie API (in format html, tex si man) pentru un proiect cu referinte incrucisate la librariile folosite.
    • Suport de internationalizare, permitand translatorilor sa isi adauge limba lor tinta intr-un proiect cu usurinta, incluzand suport pentru KBabel.
    • Suport pentru proiecte Qt/Embedded (pentru Zaurus si platforme iPAQ).
  • Suport de limbaj
    • 15 limbaje suportate incluzand Ada, C, C++, Objective-C (via C support), SQL, Fortran, Haskell, Java, PHP, Pascal, Perl, Python, Ruby, Bash, XUL (neoficial).
    • Raportor de probleme care raporteaza diferite "probleme" cu codul sursa in timp ce scrii. Problemele includ erori de sintaxa, TODO's si FIXME's.
    • Browser de clase pentru a afisa ierarhia claselor si alte simboluri din proiect.
    • Analizatoare de limbaj pentru a ajuta browser-ul de clase si raportorul de probleme.
    • Wizard-uri de clase, metode si atribute pentru limbaje orientate pe obiect.
    • Completare de cod automata si sugestii pentru variabile de clase, metode, argumente de functii si altele.
    • Formatarea sursei, sublinierea sintaxei si impaturirea codului.
    • Optiuni specifice C++, cum ar fi
      • "make member" - creaza definitia metodei in fisierul sursa bazandu-se pe declararea metodei in fisierul header;
      • "extract interface"- extrage interfata (declararea metodelor) si creaza clasa de baza abstracta cu acea interfata;
      • "subclassing" - creaza automat subclase si implementeaza slot-uri pentru formulare Qt Designer.
  • Debugger integrat
    • Frontend-uri pentru debuger-e GDB si Java.
    • Consola GDB integrata.
    • Breakpoint-uri de linie, adresa si metoda si watchpoint-uri.
    • Breakpoint-uri conditionale.
    • Disassembler si vizualizator de stiva a cadrului.
    • Watch tree pentru variabile.
    • Vizualizatoare de memorie, registri si librarii.
    • Suport pentru consola de debugger (libtool).
    • Debugging de la distanta.
  • Editarea codului
    • Orice editor care implementeaza interfata KDE KTextEditor poate fi folosit cu KDevelop.
    • Editoare disponibile sunt Kate, QEditor (bazat pe Qt Designer editor) si KVim.
    • Bookmark-uri de proiect care sunt stocate in fisierul proiectului.
    • Vizualizator de diferente intre textul editat curent si un text de pe disc.
    • Vizualizator de diferente pentru fisiere patch.
    • Integrare Grep - cautare de expresii in proiect.
    • Cautare si inlocuire in proiect.
  • Sistem de control a versiunii
    • Un API obisnuit care include suport VCS in KDevelop.
    • Sisteme VCS suportate: CVS, Perforce, Subversion si Clearcase.
  • Wizard de aplicatii
    • Wizard-ul de aplicatii genereaza modele de aplicatii complete.
    • Modele pentru fiecare limbaj si sistem de construire suportate.
    • Abilitatea de a folosi modele customizate de aplicatii care nu sunt incluse in distributie.
    • O gramada de modele de proiecte legate de KDE (aplicatii KPart, aplicatii simple de KDE, module KControl, applet-uri Kicker (panel), parti KOffice, KIOSlaves, Konqueror, Noatun si pluginuri Kate si stiluri de desktop).
  • Vizualizator de documentatie
    • O interfata unificata pentru a vedea documentatia.
    • Capabilitati TOC, index si cautare full text (cu htdig).
    • Autodetectare si incluziune pentru documentatie Qt, KDE Libs si DevHelp.
    • Suport pentru colectii de documentatie customizate (format KDevelopTOC).
    • Abilitate de a deschide documentatie locala, de la distanta si chiar compresata (folosind orice protocol suportat de Konqueror - http, ftp, sftp, tar, zip, etc.).
    • Abilitate de a deschide formate diferite de documentatie (html, text, ps, pdf, chm, etc.).
  • Alte trasaturi
    • Interfata cu utilizatorul este foarte configurabila, permite trecerea imediata intre moduri clasice MDI ca toplevel, childframe si noul mod IDEAl cu utilitar de vedere "sliding".
    • Consola inclusa.
    • Selector de fisiere, file tree si facilitati de grupuri de fisiere fara a avea nevoie de un file manager extern.
    • Includerea a oricarui alt program de care ai nevoie prin adaugarea sa la meniul "Tools" in functie de nevoile tale.
    • Tester de expresii obisnuite.
    • Integrare Valgrind pentru a gasi scurgeri de memorie in programe.
    • Utilitar de distributie si impachetare a proiectului pentru crearea automate de pachete sursa si binare.
    • Integrare CTags.
    • Explorer de parti pentrui a cunoaste serviciile KDE disponibile.
