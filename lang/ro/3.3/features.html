Conţine toate facilităţile KDevelop 3.2.x plus:
  • Noi facilităţi în secţiunea KDevelop din Planificarea facilităţilor KDE 3.5
  • Se poate folosi CMAKE pentru compilarea KDevelop
  • Suportul C++ îmbunătăţit:
    • suport pentru QT4, cod diferit generat pentru proiectele Qt3.x şi Qt4.x
    • Adăugat un nou tab în dialogul de opţiuni C++ pentru a configura biblioteca Qt folosită. Se poate configura dacă Qt e folosit, care versiune majoră şi unde este instalată biblioteca Qt.
  • Depanatorul pentru C, C++ şi Ruby îmbunătăţit:
    • Fereastra cu punctele de întrerupere este mai uşor de folosit
    • Se arată automat fereastra de variabile când aplicaţia este oprită, şi este ascunsă cînd aplicaţia se încheie.
    • Implementată editarea valorilor expresiilor/variabilelor în fereastra de variabile.
    • Îmbunătăţită vizualizarea variabilelor
  • Suport pentru limbajul PHP5
  • Un nou manager de proiect:
    • E bazat pe arhitectura platformei KDevelop unde interfaţa grafică de management a proiectului este abstractizată pentru un sistem de construire specific. Sistemele de construire sunt implementate prin plugin-uri ("importatori de proiecte, constructori şi editoare").
    • Se oferă suport pentru spaţiu de lucru (workspace).
    • Constructori de proiect: constructor 'make'
    • Importare de proiecte: automake, makefile, qmake
  • mergeţi la pagina cu schimbările pentru mai multe detalii
