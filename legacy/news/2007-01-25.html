KDevelop 3.4.0 public release.

The first major release in over a year closes more than 500 bugs. There is an impressive list of additional features including improved Qt 4 support, new debugging abilities, more attractive default user interface layout and improvements for C++, Ruby and PHP support. The most important features are highlighted in the little showcase (PDF format or ODP format). Source code and packages are available for Kubuntu and openSUSE with unofficial builds for several others on the download page.
