Binary snapshot of the stable KDevelop with a critical bugfix

It seams that the official KDevelop 3.5.1 release had a critical bug: Compiler errors were not shown in output window. But thanks to an user that posted reproducible steps on how to trigger the bug, the KDevelop team was able to fix it and release binary packages with the fix within hours of the bug report. This proves that reporting bugs correctly, allows us to fix them faster.
