Sources for KDE >= :

Latest development, and therefore unstable, snapshot (tar.bz2)

  1. Before continuing please check if your computer has the required software installed.

  2. Download the source tar.bz2 file.

  3. Uncompress the source file and change to the base directory:
    $ bunzip2 kdevelop.tar.bz2
    $ tar -xf kdevelop.tar
    $ cd kdevelop
