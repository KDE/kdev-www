This page concentrates all information about KDevelop HEAD

KDevelop HEAD contains all the most recent changes performed by the developers. That means that it changes constantly around the clock. You should only use this version if you are familiar with SVN and are wiling to update often. HEAD will give you the most features, but it can be instable sometimes.
