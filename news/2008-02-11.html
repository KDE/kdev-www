Binary snapshot of the stable KDevelop 3.5.x release series

Eight binary release of the stable KDevelop 3.5.x, probably the best IDE for Linux, Unix, Solaris and MacOSX. It supports: Ada, Bash, C/C++, Fortran, Haskell, Java, Pascal, Perl, PHP, Python, Ruby and SQL programming languages.

For a detailed list of changes see the detailed changelog. Fixes 9 bugs from KDevelop 3.5.0.7 and previous versions. You can download it from the KDevelop 3.5.x download page.
