KDevelop Platform Libraries API Reference
The next generation KDevelop will be KDevelop 4.0. It is based on new technologies like Qt 4.3 and KDE 4.0. To make it easier for others to use the power of KDevelop to build their own, IDE-like, application we have split KDevelop into a library - the KDevelop platform - and the application itself - KDevelop. This way future applications like kile, quanta, etc can share the same base code with KDevelop and profit from code reuse. So start by reading the KDevelop Platform Libraries API Reference before coding your next application.
