Sixth binary snapshot of the stable KDevelop 3.5.x release series

Fixes 75 bugs from KDevelop 3.5.0 and previous versions. Again it requires GDB >= 6.4 but a regression was found in GDB versions 6.7.0 and 6.7.1, avoid them. It will be fixed in version 6.7.2. For a detailed list of changes see the detailed changelog. You can download it from the KDevelop 3.5.x download page courtesy of openSUSE - build service.
