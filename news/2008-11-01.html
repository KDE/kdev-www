KDevelop 3.5.3-2 binary snapshot, and KDevelop 4 Alpha 3 released
There is a new KDevelop 3.5.3 binary snapshot. All users are advised to update to this version, all known crash bugs have been fixed and it is a rock-solid and full featured C++ IDE. Today we also tagged the alpha 3 of KDevelop 4 and KDevplatform 1. The C++ support is already better than the one provided by KDevelop3, and many developers already use it daily. This is the first KDevelop version that compiles and runs natively on MS windows operating systems.
