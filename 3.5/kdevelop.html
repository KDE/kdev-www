Known bugs and issues:
  • No more MDI interface modes, the only mode available is "simpleIDEAl"
  • Qeditor and yzis are no longer supported. KATE is always used.
  • Haskell programming language is no longer supported.
  • Reported bugs in bugzilla database
To report a new bug use KDevelop's menu Help -> Report Bug... or navigate to our Bugs & Wishes page.
