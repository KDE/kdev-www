Checklist for a mediawiki update

  • Get the latest version from: http://wikipedia.sourceforge.net/
  • Read the update instructions that are included in the file you just downloaded, usualy you do not have to change anything, but just in case read the file anyway
  • put the files in the mediawiki directory and commit them
  • pay special attention to the includes/SpecialPage.php file as it has some local changes to combat Spam
  • Navigate to the Shell commands section of remview and execute the script to do the database update:
    cd ../mediawiki && /usr/bin/php maintenance/update.php
  • For security reasons navigate to the sidebar customization page and remove the entry:
    • ** randompage-url|randompage
  • Get the latest bad-behavior extension version from: http://www.ioerror.us/software/bad-behavior/bad-behavior-download/ and install it at extensions/bad-behavior2/
  • The next files have local changes that should be preserved:
    • .cvsignore
    • .htaccess
    • LocalSettings.php
    • AdminSettings.php
    • includes/SpecialPage.php
    • maintenance/deleteIdleUsers.php
    • extensions/bad-behavior2/bad-behavior-generic.php
    • extensions/bad-behavior2/bad-behavior-kdevelop-with-db.php
