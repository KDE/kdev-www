Checklist for a phorum update

The Phorum homepage has good information about it.

Get the latest version from http://dev.phorum.org/snaps/.

The only files that have KDevelop specific configuration are:

  • phorum5/topusers.php
  • phorum5/switchuser.php
  • phorum5/jambot.php
  • phorum5/jambot.pl
  • phorum5/images/rss20.gif
  • phorum5/include/db/config.php
  • phorum5/include/lang/english.php
  • phorum5/templates/kdevelop_3.1/footer.tpl
  • phorum5/templates/kdevelop_3.1/header.tpl
  • phorum5/templates/kdevelop_3.1/info.php
  • phorum5/templates/kdevelop_3.1/list.tpl
  • phorum5/templates/kdevelop_3.1/posting_messageform.tpl
  • phorum5/templates/kdevelop_3.1/register.tpl
  • phorum5/templates/kdevelop_3.1/topusers.tpl
These others do get overwritten so take care because otherwise you might end up disabling the polls mod:
  • phorum5/include/lang/english.php
  • phorum5/templates/kdevelop_3.1/footer.tpl
  • phorum5/templates/kdevelop_3.1/header.tpl
  • phorum5/templates/kdevelop_3.1/info.php
  • phorum5/templates/kdevelop_3.1/list.tpl
  • phorum5/templates/kdevelop_3.1/posting_messageform.tpl
  • phorum5/templates/kdevelop_3.1/register.tpl
That means that all the other files can be overwritten to update to a new phorum version. Usually an update does not overwrite any of the mentioned files, but I've mentioned them here just in case .... ;) After the update visit the phorum5/admin.php with a web browser to activate the update process.
