Checklist for a phpMyAdmin update

  • Get the latest version from: http://www.phpmyadmin.net/
  • Read the update instructions that are included in the file you just downloaded, usualy you do not have to change anything, but just in case read the file anyway
  • use the commands available in the bin/manual_maintenance.sh to download and install the new version
  • pay special attention to the .htpasswd, .htaccess and config.inc.php files because they should not get overwritten
