This page concentrates all information about KDevelop 3.0.x release series

Development started under the Gideon codename. The email A new IDE for a new milleneum :-) from Bernd Gehrmann describes how it all started.
KDevelop 3.0 is a stable release for KDE-3.x, with a entirely new code base, many new features (e.g. support of new programming languages, new editors). Version 3.0.0 has been released with KDE-3.2.0. KDevelop 3.0.x bugfix versions will be released together with KDE-3.2.x
